# Claude 工作台

个人 AI 辅助工作台，涵盖投资分析、教育规划、小说创作等领域。

## 目录结构

### 股市/
A 股/港股/美股分析工作区，包含个股深度分析报告、龙虎榜分析、杀猪盘检测等。

- 个股深度分析报告（如贵州茅台）
- 培训材料（个人所得税专项附加扣除培训 PPT）
- 配置的技能：deep-analysis、investor-panel、lhb-analyzer、trap-detector

### 教育分析/
子女教育规划分析，包括学区房对比、人口变化趋势、换房方案等。

- 用户背景分析
- 人口变化分析
- 江夏区雅居乐花园详细分析
- 洪山区爱家名校华城详细分析
- 换房方案分析

### 小说/
长篇小说《对门》的创作工作区。

- 完整章节（ch001-ch040）
- 人物设定与大纲
- 配图版章节与图片提示词
- 配置的技能：novel-writing

## 技术栈

- AI 助手：Claude Code
- 主要技能：deep-analysis、humanities-thesis、novel-writing、frontend-design 等
