# 中文 → Skill 映射表

**学习对象**：初一学生（零基础，12~13岁）
**要求**：所有内容通俗易懂，用生活类比，语言亲切有鼓励性

在对话中直接说中文就能调用对应的 skill。以下**触发词**为同一行的任意一种说法都能调用该 skill。

## 学习 → `/study`

| 你说（以下任意均可） | 效果 |
|----------------------|------|
| `学习` / `课程` / `课程概览` / `看看课程` / `学习内容` / `要学什么` | 展示整个 C++ 课程的全景总览 + 学习路线图 + 当前进度 |
| `学习01` ~ `学习14` / `课程01` / `看看03单元` | 展示指定单元的详细内容介绍 + 学习目标 + 进度 |
| `学习 03` | 展示第03单元的详细介绍 |

根据 `01_核心知识库/` 各单元的 README 和 `05_学习系统/` 中的进度数据，组合成课程全景或单元详情报告。
可用 Python 工具 `.claude/skills/study/overview.py` 自动生成。

## 出题 → `/exam`

| 你说（以下任意均可） | 效果 |
|----------------------|------|
| `出题` / `出试卷` / `考试` / `做测试` / `测验` / `考考我` / `单元测试` | 自动找第一个未出题的单元，生成标准化试题 |

## 批改 → `/grade`

| 你说（以下任意均可） | 效果 |
|----------------------|------|
| `批改` / `改卷` / `对答案` / `检查` / `评分` / `判卷` / `帮我改` | 询问要批改哪个单元 |
| `批改01` ~ `批改14` / `改卷02` / `对答案03` / `检查05` | 批改指定单元的试卷 |
| `批改 01 小明` / `改卷 01 小红` | 批改第01单元，学生姓名小明 |

**作答方式**：学生在试题文件 `05_学习系统/01_试题库/XX_单元.md` 中直接填写：
- 选择题 → `（ B ）` 括号中填选项
- 判断题 → `（ 1 ）` 括号中填 1（对）或 0（错）
- 填空题 → 替换 `____` 为答案
- 读程序题 → 在 `______` 处写结果

然后说"批改01"，skill 自动读文件批改：
- 答案不存文件，批改时内部生成对照
- 没填的空不算分、不扣分
- 生成成绩报告，错题自动记入错题库

可用 Python 工具 `.claude/skills/grade/grader.py` 自动提取答案、计算得分、记录错题。

## 复习 → `/review`

| 你说（以下任意均可） | 效果 |
|----------------------|------|
| `复习` / `巩固` / `错题` / `回顾` / `查漏补缺` / `练练` / `再练练` | 全面复习——分析所有错题，按优先级生成练习 |
| `复习01` ~ `复习14` / `巩固02` / `错题05` / `回顾08` / `查漏补缺03` | 只针对指定单元的错题生成练习 |

根据错题库中的记录，针对薄弱知识点出同类型巩固题。

复习卷保存在 `05_学习系统/03_复习卷/`：
- 单单元复习 → `复习_01_单元名.md`
- 全面复习 → `复习_综合_日期.md`

可用 Python 工具 `.claude/skills/review/review.py` 分析错题分布、统计薄弱知识点。
