## 示例 1：Hello World 🌍

**文件**：[code/01_hello_world.cpp](code/01_hello_world.cpp)

```cpp
#include <iostream>      // 引入输入输出功能
using namespace std;     // 使用标准命名空间

int main() {             // 主函数——程序从这里开始
    cout << "Hello, World!" << endl;   // 输出：Hello, World!
    cout << "你好，世界！" << endl;     // 输出中文也可以！
    cout << "我是C++！" << endl;        // 再来一句
    
    return 0;            // 程序结束
}
```

**运行效果**：
```
Hello, World!
你好，世界！
我是C++！
```

## 示例 2：自我介绍 🤝

**文件**：[code/02_introduce.cpp](code/02_introduce.cpp)

```cpp
#include <iostream>
using namespace std;

int main() {
    string name;          // 创建一个"字符串"盒子，用来装名字
    int age;              // 创建一个"整数"盒子，用来装年龄
    
    cout << "你好！我是电脑小助手。" << endl;
    cout << "你叫什么名字？" << endl;
    
    cin >> name;          // 用户输入名字
    
    cout << "你好，" << name << "！你今年几岁了？" << endl;
    
    cin >> age;           // 用户输入年龄
    
    cout << "哇，" << age << "岁！很高兴认识你，" << name << "！" << endl;
    
    return 0;
}
```

**运行效果**（假设用户输入"小明"和"12"）：
```
你好！我是电脑小助手。
你叫什么名字？
小明
你好，小明！你今年几岁了？
12
哇，12岁！很高兴认识你，小明！
```

## 示例 3：简单的加法计算器 ➕

**文件**：[code/03_adder.cpp](code/03_adder.cpp)

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;             // 定义两个整数变量
    
    cout << "=== 加法计算器 ===" << endl;
    cout << "请输入第一个数字：";
    cin >> a;
    
    cout << "请输入第二个数字：";
    cin >> b;
    
    int sum = a + b;      // 计算 a + b，存入 sum
    
    cout << a << " + " << b << " = " << sum << endl;
    
    return 0;
}
```

**运行效果**：
```
=== 加法计算器 ===
请输入第一个数字：10
请输入第二个数字：20
10 + 20 = 30
```

## 示例 4：多种输出方式 🎨

**文件**：[code/04_multi_output.cpp](code/04_multi_output.cpp)

```cpp
#include <iostream>
using namespace std;

int main() {
    // 方式一：一条语句输出多段内容
    cout << "姓名：" << "小明" << "，年龄：" << 12 << "岁" << endl;
    
    // 方式二：先算再输出
    cout << "5 + 3 = " << 5 + 3 << endl;
    cout << "12 - 7 = " << 12 - 7 << endl;
    cout << "4 * 6 = " << 4 * 6 << endl;
    cout << "20 / 4 = " << 20 / 4 << endl;
    
    // 方式三：输出特殊字符
    cout << "换行前\\n换行后" << endl;           // \\n 也代表换行
    cout << "用\\t可以\\t制表（对齐）" << endl;  // \\t 代表 Tab 键
    
    return 0;
}
```

**运行效果**：
```
姓名：小明，年龄：12岁
5 + 3 = 8
12 - 7 = 5
4 * 6 = 24
20 / 4 = 5
换行前
换行后
用\\t可以\\t制表（对齐）
```

## 示例 5：看图猜数字 🎯

**文件**：[code/05_guess_setup.cpp](code/05_guess_setup.cpp)

```cpp
#include <iostream>
using namespace std;

int main() {
    int guess;
    
    cout << "=== 猜数字游戏 ===" << endl;
    cout << "我想了一个 1~10 之间的数字。" << endl;
    cout << "你能猜到是多少吗？" << endl;
    cout << "（提示：这次答案是 7）" << endl;
    cout << "请输入你的猜测：";
    
    cin >> guess;
    
    cout << "你猜的是：" << guess << endl;
    cout << "正确答案是：7" << endl;
    
    if (guess == 7) {
        cout << "🎉 猜对了！你好厉害！" << endl;
    } else {
        cout << "😅 猜错了，下次加油！" << endl;
    }
    
    return 0;
}
```

**运行效果**（假设猜 5）：
```
=== 猜数字游戏 ===
我想了一个 1~10 之间的数字。
你能猜到是多少吗？
（提示：这次答案是 7）
请输入你的猜测：5
你猜的是：5
正确答案是：7
😅 猜错了，下次加油！
```

## 运行这些代码的方法 🏃

### 方法一：在 Dev-C++ 中运行

1. 打开 Dev-C++
2. 点击菜单 `文件 → 新建 → 源代码`
3. 把代码**完整复制**进去
4. 点击工具栏的 **"编译运行"** 按钮（或按 `F11`）
5. 弹出来的黑窗口就是程序运行结果！

### 方法二：直接打开文件

1. 在文件夹中找到 `.cpp` 文件
2. 双击 → 自动在 Dev-C++ 中打开
3. 点击 **"编译运行"** 按钮

> ⭐ **建议**：不要直接复制运行——**自己动手敲一遍**！打字过程会让你记得更牢。
