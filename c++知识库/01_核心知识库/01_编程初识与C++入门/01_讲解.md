## 1. 什么是编程？🤔

### 生活中的例子

想象一下，你给妈妈写了一张"购物清单"：

```
1. 去超市
2. 买 3 个苹果
3. 买 1 瓶牛奶
4. 回家
```

**编程**就是给计算机写"购物清单"——只不过计算机的"语言"和我们不一样，需要用它能懂的方式告诉它做什么。

### 程序和编程语言

| 概念 | 比喻 | 说明 |
|------|------|------|
| **程序** | 一份"菜谱" | 告诉计算机每一步做什么的指令集合 |
| **编程语言** | 中文/英文 | 和计算机交流的工具 |
| **C++** | 一种"超能力语言" | 能写出很快、很强大的程序 |

### 计算机的思维

计算机很**听话**，但也很**死板**：
- ✅ 你说"把 1+1 算出来" → 它乖乖算
- ❌ 你说"大概、差不多" → 它不懂
- ❌ 你说"把那个拿过来"（没说是哪个）→ 它懵了

> 🎯 **编程的核心**：把你想做的事情，**一步一步、清清楚楚**地告诉计算机。

### C++ 能做什么？

- 🎮 **游戏**：很多游戏（如 Minecraft）是用 C++ 写的
- 🏢 **软件**：Chrome 浏览器、Office、Photoshop 都用到了 C++
- 🚗 **汽车/机器人**：很多智能设备的"大脑"用 C++ 编程
- 💻 **信息学奥赛**：NOI（全国青少年信息学奥林匹克竞赛）指定语言就是 C++

## 2. 搭建开发环境 🛠️

### 什么是 IDE？

IDE 就像编程的"Word"——一个写代码、运行代码全搞定的工具。

### 推荐：Dev-C++（最适合新手）

> **为什么选 Dev-C++？** 安装简单、体积小、开箱即用，很多信息学奥赛选手入门都用它。

### 下载安装步骤

1. 浏览器打开搜索"Dev-C++ 下载"
2. 下载最新版本（推荐 bloodshed 或 Orwell 版本）
3. 双击安装，一路点"下一步"（Next）
4. 安装完成后打开 Dev-C++

## 3. 第一个 C++ 程序 — Hello World! 🌍

### 代码模板

每个 C++ 程序都有一个"基本框架"：

```cpp
#include <iostream>   // 1️⃣ 引入工具箱（输入输出）
using namespace std;  // 2️⃣ 告诉程序用标准名字

int main() {          // 3️⃣ 主函数——程序的入口
                      //    计算机从这里开始执行
    // 你的代码写在这里 ✏️
    
    return 0;         // 4️⃣ 告诉计算机"程序正常结束"
}
```

### 逐行解释

| 行 | 说明 | 类比 |
|----|------|------|
| `#include <iostream>` | 引入"输入输出"功能 | 像做饭前先拿出锅和铲子 |
| `using namespace std;` | 告诉程序用标准名字 | 像说"我们说普通话" |
| `int main()` | 程序从这里开始运行 | 像故事的开头"从前有座山" |
| `{ }` | 大括号包起来的都是要执行的指令 | 像菜谱的步骤列表 |
| `return 0;` | 程序正常结束 | 像说"做完了，收工！" |

### 输出文字：cout

```cpp
cout << "你好，世界！" << endl;
```

| 符号 | 含义 |
|------|------|
| `cout` | 告诉计算机"把这个显示出来" |
| `<<` | 把后面的内容"送"到 cout |
| `"你好，世界！"` | 要显示的文字（**必须用双引号**） |
| `endl` | 换行（end line） |

> 🔑 **记住口诀**：`cout << 你要说的话 << endl;`

## 4. 输入数据：cin ⌨️

### 什么是输入？

程序运行后，让你**打字告诉它一些信息**——这就是输入。

```cpp
int age;               // 准备一个"盒子"叫 age，用来装年龄
cin >> age;            // 你输入一个数字，存进 age 这个盒子里
cout << age << endl;   // 把盒子里的数字显示出来
```

### cin 的用法

| 符号 | 含义 |
|------|------|
| `cin` | 从键盘读取输入 |
| `>>` | 把输入"送"进变量（注意方向！） |

> 🆚 **对比记忆**：
> - `cout <<` → 输出 → 箭头向外（内容从程序到屏幕）
> - `cin >>` → 输入 → 箭头向内（内容从键盘到程序）

### 输入多个数据

```cpp
int a, b;
cin >> a >> b;          // 先输入 a，再输入 b（中间用空格或回车隔开）
```

## 5. 注释 — 给代码加"便利贴" 📝

注释是写给**人看**的说明，计算机会**忽略**它们。

```cpp
// 这是单行注释，两个斜杠后面都是注释

/*
   这是多行注释
   可以写好几行
*/
```

> 🌟 **好习惯**：刚开始学，多写注释帮助自己理解！

## 6. 常见错误（别怕！）⚠️

### 新手最容易犯的错

| 错误 | 正确 | 原因 |
|------|------|------|
| `cout << "你好"` 缺 `;` | `cout << "你好";` | C++ 每条语句结尾必须有 **分号** |
| `int main()` 写成 `int mian()` | `int main()` | main 是固定的单词 |
| `cout << "hello"` 用了中文引号 | `cout << "hello"` | 符号必须是**英文** |
| `#include` 漏了 `#` | `#include` | `#` 很重要！ |

> 💪 **记住**：有错误不可怕！看看编译器提示的错误信息，一行一行检查，你很快就能学会自己"找bug"！
