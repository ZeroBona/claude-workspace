# 第一单元：编程初识与 C++ 入门 🚀

## 本单元学习目标

- ✅ 了解什么是"编程"和"程序"
- ✅ 知道 C++ 是什么、能做什么
- ✅ 安装并配置开发环境（IDE）
- ✅ 写出第一个 C++ 程序 —— **Hello World!**
- ✅ 学会使用 `cout` 输出文字
- ✅ 学会使用 `cin` 输入数据

## 前置要求

- 会操作电脑（创建文件夹、打字）
- 不需要任何编程基础！

## 预计学习时间

**约 60~90 分钟**
