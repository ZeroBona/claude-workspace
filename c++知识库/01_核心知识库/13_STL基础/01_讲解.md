# STL 基础 — 讲解 📖

---

## 1. 什么是 STL？🤔

**STL** = Standard Template Library（标准模板库）

C++ 自带的一套"工具箱"，装好了各种常用的数据结构和算法。

| 你需要... | 不用 STL | 用 STL |
|-----------|---------|--------|
| 动态数组 | 自己用指针+new管理 | `vector` |
| 排序 | 自己写冒泡/选择排序 | `sort()` |
| 栈 | 自己用数组模拟 | `stack` |
| 队列 | 自己写 | `queue` |

> 🌟 **STL 三部曲**：
> 1. `#include` 对应的头文件
> 2. 创建容器对象
> 3. 调用成员函数

## 2. vector — 动态数组 📦

### 为什么需要 vector？

普通数组大小固定，vector 可以**自动增长**。

### 基本用法

```cpp
#include <vector>

vector<int> v;           // 空的 vector
vector<int> v(5);        // 大小为5，初始值为0
vector<int> v(5, 3);     // 大小为5，全部初始化为3
vector<int> v = {1,2,3}; // 用初始值列表
```

### 常用操作

```cpp
vector<int> nums;

nums.push_back(10);      // 在末尾添加一个元素
nums.push_back(20);
nums.push_back(30);

cout << nums.size();     // 3（元素个数）
cout << nums[0];         // 10（像数组一样用下标）
cout << nums.front();    // 第一个元素
cout << nums.back();     // 最后一个元素

nums.pop_back();         // 删除最后一个元素
nums.clear();            // 清空所有元素
nums.empty();            // 判断是否为空
```

### 遍历

```cpp
// 方式1：下标
for (int i = 0; i < nums.size(); i++) {
    cout << nums[i] << " ";
}

// 方式2：范围 for（C++11）
for (int x : nums) {
    cout << x << " ";
}
```

## 3. sort — 排序 🎯

### 头文件

```cpp
#include <algorithm>
```

### 基本用法

```cpp
vector<int> v = {5, 2, 8, 1, 9};

sort(v.begin(), v.end());           // 从小到大排序
// v = {1, 2, 5, 8, 9}

sort(v.begin(), v.end(), greater<int>());  // 从大到小排序
// v = {9, 8, 5, 2, 1}
```

### 对数组排序

```cpp
int arr[] = {5, 2, 8, 1, 9};
int n = sizeof(arr) / sizeof(arr[0]);

sort(arr, arr + n);                 // 从小到大
```

## 4. stack — 栈 📚

### 什么是栈？

"后进先出"（Last In First Out）——像一摞书，**后放上去的先拿走**。

```cpp
#include <stack>

stack<int> s;

s.push(10);      // 入栈（放上书）
s.push(20);
s.push(30);
// 现在栈：30(顶) 20 10(底)

cout << s.top();    // 30（栈顶元素）
s.pop();            // 出栈（拿走最上面的）
cout << s.top();    // 20
cout << s.size();   // 2（元素个数）
cout << s.empty();  // false（是否为空）
```

## 5. queue — 队列 🏪

### 什么是队列？

"先进先出"（First In First Out）——像排队，**先来的先服务**。

```cpp
#include <queue>

queue<int> q;

q.push(10);      // 入队（排队）
q.push(20);
q.push(30);
// 现在队列：10(前) 20 30(后)

cout << q.front();  // 10（队首）
cout << q.back();   // 30（队尾）
q.pop();            // 出队（最前面的走了）
cout << q.front();  // 20
```
