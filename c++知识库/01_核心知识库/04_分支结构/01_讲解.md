# 分支结构 — 讲解 📖

---

## 1. 什么是分支结构？🤔

### 生活中的选择

每天你都在做选择：

```
如果 明天不下雨 → 去踢足球
否则            → 在家打游戏
```

```
如果 考试≥90分  → 奖励一个大礼物
否则如果≥80分   → 奖励一个小礼物
否则            → 鼓励一下，下次努力
```

**分支结构**就是让程序也会"做选择"——根据不同的条件，执行不同的代码。

### 程序流程

```
          ┌─────────────┐
          │  程序开始    │
          └──────┬──────┘
                 │
          ┌──────▼──────┐
          │  遇到条件判断 │
          └──────┬──────┘
                 │
        ┌────────┴────────┐
        ▼                  ▼
   ┌──────────┐      ┌──────────┐
   │ 条件为真  │      │ 条件为假  │
   │ 执行A代码 │      │ 执行B代码 │
   └────┬─────┘      └────┬─────┘
        └────────┬────────┘
                 ▼
          ┌─────────────┐
          │  继续往下走  │
          └─────────────┘
```

## 2. if 语句 — 单一选择 ✅

### 语法

```cpp
if (条件) {
    // 条件为 true 时执行的代码
}
```

### 流程图

```
       ┌───────────┐
       │  条件判断  │
       └─────┬─────┘
             │
       真 ┌──┴──┐ 假
          ▼     │
     ┌──────┐   │
     │ 执行  │   │
     └──┬───┘   │
        └──┬────┘
           ▼
       继续执行
```

### 示例

```cpp
int score;
cin >> score;

if (score >= 60) {
    cout << "恭喜，及格了！" << endl;
}
// 如果 score < 60，上面这个大括号里的代码直接跳过
```

---

## 3. if-else 语句 — 双向选择 ⚖️

### 语法

```cpp
if (条件) {
    // 条件为 true 时执行
} else {
    // 条件为 false 时执行
}
```

### 示例

```cpp
int age = 15;

if (age >= 18) {
    cout << "你成年了！" << endl;
} else {
    cout << "你还未成年" << endl;
}
```

> 💡 **生活类比**：
> ```
> if (今天下雨) {
>     带伞出门；
> } else {
>     不带伞；
> }
> ```

---

## 4. if-else if-else — 多向分支 🎯

### 语法

```cpp
if (条件1) {
    // 条件1为真时执行
} else if (条件2) {
    // 条件1为假且条件2为真时执行
} else if (条件3) {
    // 条件1、2为假且条件3为真时执行
} else {
    // 以上所有条件都为假时执行
}
```

### 示例：成绩等级

```cpp
int score;
cout << "请输入成绩：";
cin >> score;

if (score >= 90) {
    cout << "优秀！" << endl;
} else if (score >= 80) {
    cout << "良好" << endl;
} else if (score >= 70) {
    cout << "中等" << endl;
} else if (score >= 60) {
    cout << "及格" << endl;
} else {
    cout << "不及格，加油！" << endl;
}
```

> 🌟 **重要规则**：
> - if-else if 链按**从上到下**的顺序检查
> - 一旦找到**第一个为真的条件**，执行完对应的代码就**跳出整条链**
> - 后面的条件即使为真也不会执行了

### 执行过程

```
输入：85

┌─ if (85>=90) → false，继续往下
├─ else if (85>=80) → true ✅ → 输出"良好"，跳过后面所有
├─ else if (85>=70) → 不检查了
├─ else if (85>=60) → 不检查了
└─ else → 不执行了
```

## 5. 嵌套 if（if 里面套 if）🎭

### 什么情况用嵌套？

有时候一个条件判断还不够，需要在条件成立后**再判断另一个条件**。

```cpp
int age;
bool hasTicket;

cout << "请输入年龄：";
cin >> age;
cout << "有票吗？（1=有，0=没有）：";
cin >> hasTicket;

if (age >= 10) {
    if (hasTicket) {
        cout << "🎉 请进！" << endl;
    } else {
        cout << "🚫 请先去买票" << endl;
    }
} else {
    cout << "🚫 年龄不够，不能入场" << endl;
}
```

### 嵌套的流程图

```
          ┌─────────────┐
          │  age >= 10?  │
          └──────┬──────┘
            真   │   假
          ┌──────▼──┐   │
          │有票吗？   │   ▼
          └──┬──┬───┘ ┌──────┐
          真 │  │ 假  │禁止入内│
            ▼  │  ▼   └──────┘
         ┌───┐ │ ┌───┐
         │请进│ │ │买票│
         └───┘ │ └───┘
               │
```

> ⚠️ **注意**：嵌套层数不要太深（一般不超过 3 层），否则代码会变得很难读懂。

## 6. switch 语句 🔀

### 什么时候用 switch？

当你要判断的**是一个变量的多个固定值**（比如星期几、菜单选项）时，`switch` 比 `if-else if` 更清晰。

### 语法

```cpp
switch (表达式) {
    case 值1:
        // 表达式 == 值1 时执行
        break;
    case 值2:
        // 表达式 == 值2 时执行
        break;
    case 值3:
        // 表达式 == 值3 时执行
        break;
    default:
        // 所有 case 都不匹配时执行
        break;
}
```

### 示例：星期翻译

```cpp
int day;
cout << "请输入星期几（1-7）：";
cin >> day;

switch (day) {
    case 1:
        cout << "星期一" << endl;
        break;
    case 2:
        cout << "星期二" << endl;
        break;
    case 3:
        cout << "星期三" << endl;
        break;
    case 4:
        cout << "星期四" << endl;
        break;
    case 5:
        cout << "星期五" << endl;
        break;
    case 6:
        cout << "星期六" << endl;
        break;
    case 7:
        cout << "星期日" << endl;
        break;
    default:
        cout << "输入错误！请输入1-7" << endl;
        break;
}
```

### break 是什么？

`break` 告诉程序"执行完了，跳出 switch 语句"。

**如果不写 break 会发生什么？** → 会继续往下执行下一个 case！（这叫"穿透"）

```cpp
int n = 2;
switch (n) {
    case 1:
        cout << "一";
        // 没写break！继续往下走
    case 2:
        cout << "二";   // 会从这里开始执行
        // 也没写break！
    case 3:
        cout << "三";   // 继续执行到这里
        break;          // 终于遇到break了
}
// 输出："二三"
```

> 💡 **有时候故意利用穿透**：
> ```cpp
> int month = 2;
> switch (month) {
>     case 1:
>     case 3:
>     case 5:
>     case 7:
>     case 8:
>     case 10:
>     case 12:
>         cout << "31天" << endl;
>         break;
>     case 4:
>     case 6:
>     case 9:
>     case 11:
>         cout << "30天" << endl;
>         break;
>     case 2:
>         cout << "28或29天" << endl;
>         break;
> }
> ```

### switch 和 if 怎么选？

| 场景 | 推荐用 | 原因 |
|------|--------|------|
| 判断一个变量的**多个固定值** | `switch` | 更清晰、易读 |
| 判断**范围**（如 score >= 60） | `if` | switch 不能写范围 |
| 判断**复杂条件**（如 && ||） | `if` | switch 只能==比较 |
| 条件**少于 3 个** | `if` | 简单直接 |

## 7. 条件运算符（三元运算符）❓

### 简洁版 if-else

```cpp
// 标准写法
int max;
if (a > b) {
    max = a;
} else {
    max = b;
}

// 三元运算符写法——一行搞定！
int max = (a > b) ? a : b;
```

### 语法

```cpp
条件 ? 值1 : 值2
// 条件为真 → 整个表达式等于 值1
// 条件为假 → 整个表达式等于 值2
```

```cpp
string result = (score >= 60) ? "及格" : "不及格";
cout << result << endl;
```

> ⚠️ **初学建议**：三元运算符虽然简洁，但**不要嵌套使用**（如 `a?b?c:d:e`），会很难读懂。建议只在简单赋值场景使用。
