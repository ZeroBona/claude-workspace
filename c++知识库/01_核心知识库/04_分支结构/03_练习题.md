# 练习题 📝

## ⭐ 基础题

### 第1题：正负数判断
编写程序，输入一个整数，判断它是正数、负数还是零。

**运行效果**：
```
请输入一个整数：-5
-5 是负数

请输入一个整数：0
0 既不是正数也不是负数
```

### 第2题：最大数
编写程序，输入两个整数，输出较大的那个。

**运行效果**：
```
请输入两个整数：15 28
较大的数是：28
```

### 第3题：星期翻译
用 switch 语句，输入数字 1-7，输出对应的星期英文：
```
1→Monday 2→Tuesday 3→Wednesday 4→Thursday
5→Friday 6→Saturday 7→Sunday
```

### 第4题：三个数排序 🏆

**运行效果**：

请输入三个整数：7 3 9

### 第5题：闰年判断器 📅
编写程序，输入年份，判断是否是闰年。

> **闰年规则**：
> - 能被 400 整除的是闰年
> - 能被 4 整除但不能被 100 整除的是闰年
> - 其他都不是闰年

**运行效果**：
```
请输入年份：2024
2024年是闰年 ✅

请输入年份：2025
2025年不是闰年 ❌
```

### 第6题：BMI 健康指数 🏃
输入身高（米）和体重（公斤），计算 BMI 并判断体型。

> **公式**：BMI = 体重 ÷ (身高 × 身高)
>
> | BMI 范围 | 判断结果 |
> |----------|---------|
> | BMI < 18.5 | 偏瘦 |
> | 18.5 ≤ BMI < 24 | 正常 |
> | 24 ≤ BMI < 28 | 偏胖 |
> | BMI ≥ 28 | 肥胖 |

**运行效果**：
```
请输入身高（米）：1.65
请输入体重（公斤）：55
BMI = 20.2，体型：正常 ✅
```

## ⭐⭐⭐ 挑战题

### 第7题：猜拳游戏 ✊✋✌️
编写程序，用户输入 1（石头）/ 2（剪刀）/ 3（布），电脑也随机出一个，判断输赢。

> **提示**：先用 `srand(time(0))` 初始化随机数，再用 `rand() % 3 + 1` 生成 1-3 的随机数
> 记得 `#include <cstdlib>` 和 `#include <ctime>`

**运行效果**：
```
=== 石头剪刀布 ===
1: 石头 ✊
2: 剪刀 ✌️
3: 布 ✋
请选择：1
你出：石头 ✊
电脑出：剪刀 ✌️
🎉 你赢了！
```

### 第8题：日期合法性判断 📆
输入年、月、日，判断这个日期是否合法。

> **提示**：需要考虑各月的天数差异、闰年2月有29天等

**运行效果**：
```
请输入年 月 日：2024 2 29
2024年2月29日 ✅ 合法日期

请输入年 月 日：2023 2 29
❌ 不合法日期！2023年不是闰年，2月只有28天
```

### 第7题
```cpp
#include <iostream>
#include <cstdlib>
#include <ctime>
using namespace std;

int main() {
    srand(time(0));

    cout << "=== 石头剪刀布 ===" << endl;
    cout << "1: 石头 ✊" << endl;
    cout << "2: 剪刀 ✌️" << endl;
    cout << "3: 布 ✋" << endl;

    int player, computer;
    cout << "请选择：";
    cin >> player;
    computer = rand() % 3 + 1;

    cout << "你出：";
    if (player == 1) cout << "石头 ✊" << endl;
    else if (player == 2) cout << "剪刀 ✌️" << endl;
    else cout << "布 ✋" << endl;

    cout << "电脑出：";
    if (computer == 1) cout << "石头 ✊" << endl;
    else if (computer == 2) cout << "剪刀 ✌️" << endl;
    else cout << "布 ✋" << endl;

    if (player == computer) {
        cout << "🤝 平局！" << endl;
    } else if ((player == 1 && computer == 2) ||
               (player == 2 && computer == 3) ||
               (player == 3 && computer == 1)) {
        cout << "🎉 你赢了！" << endl;
    } else {
        cout << "😅 你输了！" << endl;
    }

    return 0;
}
```
