# 循环结构 — 讲解 📖

---

## 1. 为什么需要循环？🤔

### 没有循环的痛苦

假设你要输出 1 到 100：

```cpp
cout << 1 << endl;
cout << 2 << endl;
cout << 3 << endl;
// ... 你要写100行？？？
cout << 100 << endl;
```

这太累了！而且如果换成 1 到 10000 呢？

### 用循环多轻松

```cpp
int i = 1;
while (i <= 100) {
    cout << i << endl;
    i++;
}
```

> 🌟 **循环的核心价值**：**用很少的代码，做很多次重复的事情**

### 生活中的循环

| 场景 | 循环条件 | 每次做的事 |
|------|---------|-----------|
| 跑圈 | 还没跑完5圈 | 跑一圈 |
| 背单词 | 还没背完今天任务 | 背一个单词 |
| 检查作业 | 还有没检查的题 | 检查一题 |

## 2. while 循环 🏃

### 语法

```cpp
while (条件) {
    // 条件为 true 时反复执行
}
```

### 流程图

```
       ┌───────────┐
       │  初始化变量 │
       └─────┬─────┘
             │
       ┌─────▼──────┐
       │ while(条件) │──── 假 ───→ 继续往后
       └─────┬──────┘
             │ 真
       ┌─────▼──────┐
       │  循环体代码  │
       │  (别忘了让  │
       │   条件变化)  │
       └─────┬──────┘
             │
             └──→ 回到 while(条件)
```

### 示例：数数 1 到 5

```cpp
int i = 1;           // 1️⃣ 初始化（从1开始）

while (i <= 5) {     // 2️⃣ 条件（i <= 5 就继续）
    cout << i << " "; // 3️⃣ 循环体（输出 i）
    i++;              // 4️⃣ 更新（i 加 1）
}
// 输出：1 2 3 4 5
```

**一步步执行过程**：
```
第1轮：i=1，1<=5? 真 → 输出1，i变成2
第2轮：i=2，2<=5? 真 → 输出2，i变成3
第3轮：i=3，3<=5? 真 → 输出3，i变成4
第4轮：i=4，4<=5? 真 → 输出4，i变成5
第5轮：i=5，5<=5? 真 → 输出5，i变成6
第6轮：i=6，6<=5? 假 → 循环结束
```

### ⚠️ 无限循环——小心！

```cpp
int i = 1;
while (i <= 5) {
    cout << i << " ";
    // 忘记写 i++ 了！！！
}
// 程序会永远输出 1 1 1 1 1 ... 停不下来！
```

> 🔑 **循环三要素**：**初始化、条件、更新**，缺一不可！

## 3. do-while 循环 🚶

### 和 while 的区别

`do-while` **先执行一次循环体**，再判断条件。

### 语法

```cpp
do {
    // 至少执行一次
} while (条件);
```

### 示例

```cpp
int i = 1;
do {
    cout << i << " ";
    i++;
} while (i <= 5);
// 输出：1 2 3 4 5（效果和 while 一样）
```

### 和 while 的核心区别

```cpp
// while：先判断，再执行
int a = 10;
while (a <= 5) {
    cout << "不会执行" << endl;
}

// do-while：先执行一次，再判断
int b = 10;
do {
    cout << "至少执行一次" << endl;
} while (b <= 5);
// 输出："至少执行一次"
```

| 场景 | 推荐 |
|------|------|
| 可能一次都不需要执行 | `while` |
| 至少需要执行一次 | `do-while` |

> 🌟 **实际开发**：`while` 用的最多，`do-while` 用的较少。
> 一个典型场景是**猜数字游戏**——至少猜一次：

```cpp
int guess;
do {
    cout << "猜一个数字：";
    cin >> guess;
} while (guess != 7);

cout << "猜对了！" << endl;
```

## 4. for 循环 🎯

### 语法

```cpp
for (初始化; 条件; 更新) {
    // 循环体
}
```

### 三部分

```cpp
for (int i = 1;   i <= 5;   i++) {
    //         ↑      ↑       ↑
    //      初始化   条件    更新
    cout << i << " ";
}
```

| 部分 | 作用 | 执行时机 |
|------|------|---------|
| `初始化` | 定义循环变量 | 循环开始前执行一次 |
| `条件` | 决定是否继续 | 每次循环前判断 |
| `更新` | 改变循环变量 | 每次循环体执行完后执行 |

### 执行过程

```
1. 执行 初始化（int i = 1）
2. 判断 条件（i <= 5?）
   → 真：执行循环体 → 执行 更新（i++） → 回到第2步
   → 假：结束循环
```

### 示例：1 加到 100

```cpp
int sum = 0;

for (int i = 1; i <= 100; i++) {
    sum = sum + i;   // 或者 sum += i
}

cout << "1+2+...+100 = " << sum << endl;
// 输出：5050
```

> 🌟 **for 循环 vs while 循环**：两者能做的事完全一样，只是写法不同。
> - 明确知道要循环多少次 → **for 循环**
> - 不确定要循环多少次 → **while 循环**

## 5. break 和 continue 🚦

### break — 强行终止循环

`break` 让循环**立即结束**，不管条件是否还成立。

```cpp
// 猜数字：猜对了就结束
for (int i = 1; i <= 100; i++) {
    int guess;
    cin >> guess;

    if (guess == 7) {
        cout << "猜对了！用了" << i << "次" << endl;
        break;   // 跳出循环，不等了！
    }
}
```

### continue — 跳过本次循环

`continue` 跳过**本次循环剩余代码**，直接进入下一次。

```cpp
// 输出 1 到 10 中的偶数
for (int i = 1; i <= 10; i++) {
    if (i % 2 != 0) {
        continue;   // 奇数就跳过，不输出
    }
    cout << i << " ";   // 只输出偶数
}
// 输出：2 4 6 8 10
```

### 对比

| 指令 | 效果 | 比喻 |
|------|------|------|
| `break` | **整个循环**结束 | 不想跑了，直接不跑了 |
| `continue` | **本次跳过**，下次继续 | 这圈累了歇歇，下圈继续跑 |

## 6. 嵌套循环 🔄🔄

### 循环里面套循环

```cpp
for (int i = 1; i <= 3; i++) {        // 外层循环
    for (int j = 1; j <= 3; j++) {    // 内层循环
        cout << i << "," << j << "  ";
    }
    cout << endl;   // 每行结束换行
}
```

**执行过程**：
```
i=1 → j=1,2,3 → 输出"1,1  1,2  1,3"
i=2 → j=1,2,3 → 输出"2,1  2,2  2,3"
i=3 → j=1,2,3 → 输出"3,1  3,2  3,3"
```

> 💡 **理解嵌套**：外层循环走一次，内层循环走**全部**。

### 经典：打印九九乘法表

```cpp
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= i; j++) {   // 注意：j <= i
        cout << j << "×" << i << "=" << i * j << "\t";
    }
    cout << endl;
}
```

**运行效果**：
```
1×1=1
1×2=2  2×2=4
1×3=3  2×3=6  3×3=9
1×4=4  2×4=8  3×4=12 4×4=16
...
```

## 7. 三种循环怎么选？🤷

| 场景 | 推荐循环 | 原因 |
|------|---------|------|
| 知道具体循环次数 | `for` | 很方便 |
| 不知道次数，只知道条件 | `while` | 灵活 |
| 至少需要执行一次 | `do-while` | 先执行后判断 |
| 遍历数组（后面学） | `for` | 下标控制方便 |

> 🌟 **初学建议**：
> 1. 大部分时候用 `for`（最容易控制）
> 2. 不确定次数用 `while`
> 3. 几乎不用 `do-while`（了解就行）
