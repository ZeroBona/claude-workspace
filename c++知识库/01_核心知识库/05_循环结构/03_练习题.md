## ⭐⭐ 进阶题

### 第4题：最大公约数（辗转相除法）🔢
输入两个正整数，用辗转相除法求它们的最大公约数。

> **辗转相除法**：
> 1. 用大数除以小数，取余数
> 2. 如果余数为0，则除数就是最大公约数
> 3. 否则，用除数÷余数，继续

**运行效果**：
```
请输入两个数：48 36
48 和 36 的最大公约数是：12
```

### 第5题：判断素数 🔍
输入一个正整数，判断它是不是素数（质数）。

> **素数**：只能被 1 和它本身整除的数（如 2,3,5,7,11...）

**运行效果**：
```
请输入一个正整数：17
17 是素数 ✅

请输入一个正整数：21
21 不是素数 ❌（可以被 3 或 7 整除）
```

### 第6题：数字反转 🔄
输入一个正整数，输出它的反转数。

**运行效果**：
```
请输入一个正整数：12345
反转后：54321
```

> **提示**：每次取个位（`%10`），然后去掉个位（`/10`），直到变成0

### 第9题：水仙花数 🌸
水仙花数是一个三位数，它的各位数字的立方和等于它本身。

例如：153 = 1³ + 5³ + 3³ = 1 + 125 + 27

找出所有三位数中的水仙花数。

**运行效果**：
```
水仙花数：153 370 371 407
```

### 第5题（素数判断）
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    bool isPrime = true;

    cout << "请输入一个正整数：";
    cin >> n;

    if (n < 2) {
        isPrime = false;
    } else {
        for (int i = 2; i * i <= n; i++) {
            if (n % i == 0) {
                isPrime = false;
                break;
            }
        }
    }

    if (isPrime) {
        cout << n << " 是素数 ✅" << endl;
    } else {
        cout << n << " 不是素数 ❌" << endl;
    }
    return 0;
}
```

### 第6题（数字反转）
```cpp
#include <iostream>
using namespace std;

int main() {
    int n, rev = 0;
    cout << "请输入一个正整数：";
    cin >> n;

    while (n > 0) {
        rev = rev * 10 + n % 10;   // 加上个位
        n /= 10;                    // 去掉个位
    }

    cout << "反转后：" << rev << endl;
    return 0;
}
```

### 第9题（水仙花数）

#include <iostream>
using namespace std;

int main() {
    cout << "水仙花数：";
    for (int n = 100; n <= 999; n++) {
        int ge = n % 10;
        int shi = n / 10 % 10;
        int bai = n / 100;

        if (bai*bai*bai + shi*shi*shi + ge*ge*ge == n) {
            cout << n << " ";
        }
    }
    cout << endl;
    return 0;
}
