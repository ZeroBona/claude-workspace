# 面向对象入门 — 讲解 📖

---

## 1. 面向过程 vs 面向对象 🤔

### 面向过程（之前学的）

```cpp
int add(int a, int b) { return a + b; }
int main() { cout << add(3, 5); }
```

核心是**函数**——把一个大任务拆成一个个函数。

### 面向对象

核心是**对象**——把数据和操作数据的方法打包在一起。

```
面向过程：          面向对象：
  函数（动作）         对象（名词）
  数据（分开的）        数据 + 动作 = 类
```

> 🌟 **生活类比**：
> - **面向过程**：做蛋糕 → ①准备材料 ②搅拌 ③烘烤（重点是步骤）
> - **面向对象**：烤箱、搅拌器、材料（重点是谁来做）
> - 对象 = "能做事的东西"

## 2. 类（class）的定义 📝

### struct 回顾

```cpp
struct Student {
    string name;     // 只有数据
    double score;
};
```

### class 的升级

```cpp
class Student {
public:                     // 公开的——外部可以访问
    // 构造函数（创建对象时自动调用）
    Student(string n, int a, double s) {
        name = n;
        age = a;
        score = s;
    }

    // 成员函数
    void sayHello() {
        cout << "大家好，我是" << name << endl;
    }

    void showInfo() {
        cout << name << " " << age << "岁 " << score << "分" << endl;
    }

private:                    // 私有的——外部不能直接访问
    string name;
    int age;
    double score;
};
```

### 使用类

```cpp
Student s1("小明", 13, 95.5);   // 创建对象（自动调用构造函数）
s1.sayHello();                   // 调用成员函数
s1.showInfo();
```

## 3. 封装 🔒

### 什么是封装？

把数据（成员变量）藏起来，只通过成员函数来操作。

```cpp
class BankAccount {
public:
    BankAccount(string n, double m) {
        name = n;
        money = m;
    }

    void save(double amount) {     // 存钱
        if (amount > 0) money += amount;
    }

    void take(double amount) {     // 取钱
        if (amount > 0 && amount <= money) money -= amount;
    }

    double getMoney() {            // 查余额
        return money;
    }

private:
    string name;
    double money;      // 私有！外部不能直接改
};

int main() {
    BankAccount acc("小明", 1000);
    acc.save(500);          // ✅ 通过函数操作
    // acc.money = 99999;   // ❌ 私有成员不能直接访问
    cout << "余额：" << acc.getMoney() << endl;
    return 0;
}
```

> 💡 **封装的好处**：
> - 保护数据不被乱改
> - 隐藏实现细节
> - 想改内部实现时，不影响外部代码

## 4. 构造函数 🏗️

### 自动调用的初始化函数

```cpp
class Clock {
public:
    // 构造函数——名字和类名相同，没有返回值
    Clock(int h, int m, int s) {
        hour = h;
        minute = m;
        second = s;
    }

    void showTime() {
        cout << hour << ":" << minute << ":" << second << endl;
    }

private:
    int hour, minute, second;
};

int main() {
    Clock c1(10, 30, 0);    // 自动调用构造函数
    c1.showTime();           // 10:30:0
    return 0;
}
```
