# 文件操作 — 讲解 📖

---

## 1. 为什么需要文件操作？🤔

### 问题

之前写的程序，数据都存在变量里——程序一关，数据就丢了！

```
程序运行 → 输入数据 → 处理 → 输出到屏幕 → 程序关闭 → 数据没了 ❌
```

### 文件读写

```
程序运行 → 输入数据 → 处理 → 保存到文件 → 下次还能用 ✅
```

> 🌟 **文件**：数据存到硬盘上，关了程序还在。

### 需要头文件

```cpp
#include <fstream>     // 文件流
```

---

## 2. 写入文件（ofstream）✏️

### 基本步骤

```cpp
#include <fstream>

ofstream fout;               // 1. 创建输出流对象
fout.open("data.txt");       // 2. 打开文件（不存在会自动创建）
fout << "Hello, File!";      // 3. 写入数据（就像 cout 一样）
fout.close();                // 4. 关闭文件
```

### 完整示例

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream fout;
    fout.open("student.txt");

    fout << "姓名：小明" << endl;
    fout << "年龄：13" << endl;
    fout << "成绩：95.5" << endl;

    fout.close();
    cout << "写入完成！" << endl;

    return 0;
}
```

> 运行后，程序所在目录会生成一个 `student.txt` 文件。

---

## 3. 读取文件（ifstream）📖

### 基本步骤

```cpp
ifstream fin;                // 1. 创建输入流对象
fin.open("data.txt");        // 2. 打开文件
string s;
fin >> s;                    // 3. 读取数据（就像 cin 一样）
fin.close();                 // 4. 关闭文件
```

### 按行读取

```cpp
ifstream fin;
fin.open("student.txt");

string line;
while (getline(fin, line)) {   // 按行读取，直到文件末尾
    cout << line << endl;
}

fin.close();
```

### 判断文件是否打开成功

```cpp
ifstream fin;
fin.open("data.txt");

if (!fin.is_open()) {        // 或者 if (fin.fail())
    cout << "文件打开失败！" << endl;
    return 1;
}

// 正常读取...
fin.close();
```

## 4. 追加写入 ➕

### 不想覆盖原有内容？

```cpp
ofstream fout;
fout.open("log.txt", ios::app);   // ios::app = append 追加模式
fout << "新增一行" << endl;
fout.close();
```

| 模式 | 作用 |
|------|------|
| 默认 | 覆盖写入 |
| `ios::app` | 追加到末尾 |
| `ios::in` | 读模式 |
| `ios::out` | 写模式 |
