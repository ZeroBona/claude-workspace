# 练习题 📝

## ⭐ 基础题

### 第1题：大小写转换
输入一个字符串，把所有小写字母转成大写，大写转成小写。

**运行效果**：
```
请输入字符串：Hello World 123
转换结果：hELLO wORLD 123
```

### 第2题：统计元音字母
输入一个字符串，统计其中的元音字母（a e i o u）个数。

**运行效果**：
```
请输入字符串：Hello C++ Programming
元音字母个数：5
```

### 第3题：字符串反转
输入一个字符串，输出它的反转。

**运行效果**：
```
请输入字符串：Hello
反转后：olleH
```

输入一个字符串和一个偏移量 k（1-25），将每个字母向后偏移 k 位。

**运行效果**：
```

请输入偏移量：3

```

### 第1题（大小写转换）
```cpp
#include <iostream>
#include <string>
#include <cctype>
using namespace std;

int main() {
    string s;
    cout << "请输入字符串：";
    getline(cin, s);

    for (int i = 0; i < s.length(); i++) {
        if (islower(s[i])) s[i] = toupper(s[i]);
        else if (isupper(s[i])) s[i] = tolower(s[i]);
    }

    cout << "转换结果：" << s << endl;
    return 0;
}
```

### 第3题（字符串反转）
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cout << "请输入字符串：";
    getline(cin, s);

    for (int i = 0; i < s.length() / 2; i++) {
        char temp = s[i];
        s[i] = s[s.length() - 1 - i];
        s[s.length() - 1 - i] = temp;
    }

    cout << "反转后：" << s << endl;
    return 0;
}
```

### 第6题（凯撒密码）
```cpp
#include <iostream>
#include <string>
#include <cctype>
using namespace std;

int main() {
    string s;
    int k;
    cout << "请输入字符串：";
    getline(cin, s);
    cout << "请输入偏移量：";
    cin >> k;

    for (int i = 0; i < s.length(); i++) {
        if (isupper(s[i])) {
            s[i] = (s[i] - 'A' + k) % 26 + 'A';
        } else if (islower(s[i])) {
            s[i] = (s[i] - 'a' + k) % 26 + 'a';
        }
    }

    cout << "加密后：" << s << endl;
    return 0;
}
```
