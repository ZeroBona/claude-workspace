# 字符串 — 讲解 📖

---

## 1. 什么是字符串？🤔

### 字符 vs 字符串

| 类型 | 内容 | 举例 | 引号 |
|------|------|------|------|
| `char` | **单个**字符 | `'A'`, `'9'`, `'$'` | **单**引号 |
| `string` | **一串**字符 | `"Hello"`, `"你好"` | **双**引号 |

### string 本质

`string` 其实就是一个**字符数组**：

```
string s = "Hello";

下标：  0   1   2   3   4
       ┌───┬───┬───┬───┬───┐
s:     │ H │ e │ l │ l │ o │
       └───┴───┴───┴───┴───┘
```

### 使用 string 需要

```cpp
#include <iostream>
#include <string>    // 可以不加，iostream 已经包含了
using namespace std;
```

## 2. 字符串的基本操作 🛠️

### 定义和初始化

```cpp
string s1;                // 空字符串
string s2 = "Hello";      // C++ 风格
string s3("Hello");       // 另一种写法
string s4 = s2;           // 复制
```

### 拼接（+ 运算符）

```cpp
string a = "Hello";
string b = "World";
string c = a + " " + b;     // "Hello World"

cout << c << endl;
// 输出：Hello World

a += " C++";                // a 变成 "Hello C++"
```

### 获取长度

```cpp
string s = "Hello";
cout << s.length() << endl;   // 5
cout << s.size() << endl;     // 5（length 的别名）
```

### 下标访问

```cpp
string s = "Hello";
cout << s[0] << endl;    // H
cout << s[1] << endl;    // e

s[0] = 'h';              // 修改：s 变成 "hello"
```

### 比较

```cpp
string a = "apple";
string b = "banana";

if (a == b) cout << "相等";
if (a != b) cout << "不相等";
if (a < b) cout << "a在前面";   // 按字典序比较
// 'a' < 'b'，所以 "apple" < "banana"
```

## 3. 输入字符串 ⌨️

### cin 的问题

```cpp
string name;
cin >> name;           // 用户输入：Xiao Ming
cout << name << endl;  // 输出：Xiao（遇到空格就停了！）
```

### getline 读取整行

```cpp
string fullName;
getline(cin, fullName);   // 可以读空格！

cout << fullName << endl; // 输出：Xiao Ming
```

### 混合输入注意事项

```cpp
int age;
string name;

cin >> age;              // 输入 13，然后按回车
// ⚠️ 回车符还留在输入缓冲区！

getline(cin, name);      // 会直接读到一个空行！

// 解决方法：用 cin.ignore() 忽略缓冲区中的回车
cin.ignore();            // 忽略换行符
getline(cin, name);      // 现在正常了
```

## 4. 遍历字符串 🔄

### 用 for + 下标

```cpp
string s = "Hello";
for (int i = 0; i < s.length(); i++) {
    cout << s[i] << " ";
}
// 输出：H e l l o
```

### 统计字符类型

```cpp
string s = "Hello123";
int letterCount = 0, digitCount = 0;

for (int i = 0; i < s.length(); i++) {
    if (s[i] >= 'a' && s[i] <= 'z') letterCount++;
    if (s[i] >= 'A' && s[i] <= 'Z') letterCount++;
    if (s[i] >= '0' && s[i] <= '9') digitCount++;
}

cout << "字母：" << letterCount << " 数字：" << digitCount << endl;
```

## 5. 常用字符串函数 📋

| 函数 | 作用 | 示例 |
|------|------|------|
| `s.length()` | 获取长度 | `s.length()` |
| `s.find(p)` | 查找子串位置 | `s.find("lo")` → 3 |
| `s.substr(pos, len)` | 提取子串 | `s.substr(1, 3)` → "ell" |
| `s.insert(pos, str)` | 插入 | `s.insert(3, "XXX")` |
| `s.erase(pos, len)` | 删除 | `s.erase(1, 2)` |
| `s.empty()` | 判断是否为空 | `s.empty()` → true/false |

### find 查找

```cpp
string s = "hello world";
int pos = s.find("world");      // pos = 6
int pos2 = s.find("xyz");       // 没找到，返回 string::npos

if (pos != string::npos) {
    cout << "找到了，位置：" << pos << endl;
}
```

### substr 提取子串

```cpp
string s = "Hello, C++!";
string sub = s.substr(7, 3);     // 从下标7开始取3个字符
cout << sub << endl;              // 输出：C++
```

## 6. 字符处理函数（需要 <cctype>）🔤

| 函数 | 作用 |
|------|------|
| `isdigit(c)` | 判断是否是数字 |
| `isalpha(c)` | 判断是否是字母 |
| `islower(c)` | 判断是否是小写 |
| `isupper(c)` | 判断是否是大写 |
| `toupper(c)` | 转大写 |
| `tolower(c)` | 转小写 |

```cpp
#include <cctype>

char c = 'a';
cout << isalpha(c) << endl;   // 1（是字母）
cout << islower(c) << endl;   // 1（是小写）
cout << (char)toupper(c) << endl;  // 'A'（转大写）
```
