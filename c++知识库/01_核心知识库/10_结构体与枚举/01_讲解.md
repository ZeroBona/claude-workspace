# 结构体与枚举 — 讲解 📖

---

## 1. 为什么需要结构体？🤔

### 问题

要存储一个学生的信息——姓名、年龄、成绩：

```cpp
string name = "小明";
int age = 13;
double score = 95.5;
```

如果**有 40 个学生**呢？要分别存 40 个名字、40 个年龄、40 个成绩数组——很麻烦！

### 结构体——"自定义数据类型"

```cpp
struct Student {         // 定义结构体
    string name;         // 成员变量
    int age;
    double score;
};

Student s1;              // 像用 int 一样用 Student
s1.name = "小明";
s1.age = 13;
s1.score = 95.5;
```

> 🌟 **结构体**把多个相关的数据打包成一个"自定义类型"。

## 2. 结构体的定义和使用 📝

### 定义

```cpp
struct 结构体名 {
    类型 成员1;
    类型 成员2;
    // ...
};
```

```cpp
struct Book {
    string title;
    string author;
    int pages;
    double price;
};
```

### 使用

```cpp
Book b1;                    // 创建结构体变量
b1.title = "C++入门";       // 用 . 访问成员
b1.author = "张三";
b1.pages = 200;
b1.price = 39.9;

cout << "书名：" << b1.title << endl;
cout << "作者：" << b1.author << endl;
```

### 初始化

```cpp
// 方式1：逐个赋值
Student s1;
s1.name = "小明";
s1.age = 13;
s1.score = 95.5;

// 方式2：花括号初始化
Student s2 = {"小红", 12, 88.5};

// 方式3：指定成员初始化（C++11）
Student s3 = {.name = "小刚", .age = 14, .score = 92.0};
```

## 3. 结构体数组 📚

```cpp
struct Student {
    string name;
    int age;
    double score;
};

Student students[3] = {      // 结构体数组
    {"小明", 13, 95.5},
    {"小红", 12, 88.5},
    {"小刚", 14, 92.0}
};

// 遍历
for (int i = 0; i < 3; i++) {
    cout << students[i].name << " "
         << students[i].age << "岁 "
         << students[i].score << "分" << endl;
}
```

**运行效果**：
```
小明 13岁 95.5分
小红 12岁 88.5分
小刚 14岁 92分
```

## 4. 结构体做函数参数 📨

```cpp
void printStudent(Student s) {        // 值传递
    cout << s.name << " " << s.age << "岁" << endl;
}

void updateScore(Student &s, double newScore) {  // 引用传递
    s.score = newScore;
}

int main() {
    Student s = {"小明", 13, 95.5};
    printStudent(s);
    updateScore(s, 99.0);
    cout << "新成绩：" << s.score << endl;
    return 0;
}
```

## 5. 枚举——给数字起名字 🔢

### 枚举解决什么问题？

```cpp
// 不用枚举——数字代表什么要用脑子记
int color = 1;   // 1=红色, 2=绿色, 3=蓝色
if (color == 1) cout << "红色";
```

### 用枚举

```cpp
enum Color {           // 定义枚举
    RED,               // 默认 = 0
    GREEN,             // = 1
    BLUE               // = 2
};

Color c = RED;         // 使用枚举
cout << c << endl;     // 输出 0

if (c == RED) {
    cout << "红色" << endl;
}
```

### 指定值

```cpp
enum Weekday {
    MON = 1, TUE, WED, THU, FRI, SAT, SUN
};

Weekday today = WED;
cout << today << endl;  // 输出 3
```

### 枚举的好处

| 不用枚举 | 用枚举 |
|---------|--------|
| `int d = 3;` | `Weekday d = WED;` |
| `if (d == 3)` | `if (d == WED)` |
| 记不住 3 是什么 | 一看 WED 就知道是周三 |
