# 函数 — 讲解 📖

---

## 1. 什么是函数？🤔

### 生活中的例子

想象你在做西红柿炒蛋：

```
菜谱：
1. 打鸡蛋 → 打3个鸡蛋，加点盐搅匀
2. 切西红柿 → 切2个西红柿
3. 炒 → 先炒鸡蛋，盛出，再炒西红柿，最后合在一起
```

如果明天要做**两次**这道菜，你不会重新写菜谱，而是**重复使用**它。

### 函数就是"可重复使用的代码块"

```cpp
// 定义一个函数（写一次）
void sayHello() {
    cout << "你好！" << endl;
}

int main() {
    sayHello();   // 调用函数（用很多次）
    sayHello();   // 再用一次
    sayHello();   // 再用一次
    return 0;
}
// 输出：
// 你好！
// 你好！
// 你好！
```

> 🔑 **函数的意义**：
> 1. **避免重复代码** → 写一次，用多次
> 2. **让代码更清晰** → 取个好名字，一看就知道功能
> 3. **方便修改** → 改一处，所有调用的地方都生效

## 2. 函数的定义和调用 📝

### 语法

```cpp
返回值类型 函数名(参数列表) {
    // 函数体
    // 函数要执行的代码
    return 返回值;   // 如果有返回值
}
```

### 最简单的函数——无参数、无返回值

```cpp
void printLine() {       // void = 没有返回值
    cout << "==========" << endl;
}

int main() {
    printLine();         // 调用
    cout << "Hello" << endl;
    printLine();         // 再调用
    return 0;
}
```

### 有参数的函数

```cpp
void printNumber(int n) {     // n 是参数
    cout << "数字：" << n << endl;
}

int main() {
    printNumber(10);          // 输出：数字：10
    printNumber(99);          // 输出：数字：99
    return 0;
}
```

### 有返回值的函数

```cpp
int add(int a, int b) {
    int sum = a + b;
    return sum;               // 把结果返回给调用者
}

int main() {
    int result = add(3, 5);   // result = 8
    cout << "3+5=" << result << endl;

    cout << "10+20=" << add(10, 20) << endl;  // 直接输出返回值
    return 0;
}
```

### 函数定义的位置

函数必须先定义（或声明）再使用：

```cpp
// ✅ 方法1：函数定义在 main 之前
int add(int a, int b) {
    return a + b;
}

int main() {
    cout << add(3, 5) << endl;
    return 0;
}
```

```cpp
// ✅ 方法2：在 main 之前声明，之后定义
int add(int a, int b);     // 声明（告诉编译器有这么一个函数）

int main() {
    cout << add(3, 5) << endl;
    return 0;
}

int add(int a, int b) {    // 定义
    return a + b;
}
```

## 3. 参数传递 📨

### 值传递（默认方式）

```cpp
void changeValue(int x) {
    x = 100;       // 修改的是 x 的副本
    cout << "函数内部：" << x << endl;   // 100
}

int main() {
    int num = 10;
    changeValue(num);
    cout << "函数外部：" << num << endl;  // 还是10！
    return 0;
}
```

> 💡 **值传递**：函数得到的是**副本**，修改副本不影响原变量。

### 引用传递（加 &）

```cpp
void changeValue(int &x) {    // 注意 &！
    x = 100;                  // 直接修改原变量
}

int main() {
    int num = 10;
    changeValue(num);
    cout << num << endl;       // 100！被修改了！
    return 0;
}
```

| 传递方式 | 符号 | 修改原变量？ | 适用场景 |
|---------|------|------------|---------|
| 值传递 | 无 | ❌ 不会 | 只读数据 |
| 引用传递 | `&` | ✅ 会 | 需要修改原数据 |

## 4. 变量的作用域 🌐

### 局部变量

在**函数内部**定义的变量，只能在这个函数里用：

```cpp
void funcA() {
    int x = 10;       // x 是 funcA 的局部变量
    cout << x << endl;
}

void funcB() {
    // cout << x << endl;  // ❌ 错误！x 在 funcA 里，这里访问不到
}

int main() {
    // cout << x << endl;  // ❌ 同样访问不到
    return 0;
}
```

### 全局变量

在**所有函数之外**定义的变量，所有地方都能用：

```cpp
int globalVar = 100;       // 全局变量

void func() {
    cout << globalVar << endl;  // ✅ 可以访问
}

int main() {
    cout << globalVar << endl;  // ✅ 可以访问
    return 0;
}
```

> ⚠️ **尽量少用全局变量**！太多全局变量会让程序难维护。能用局部变量就用局部变量。

## 5. 函数重载（Overloading）🔄

### 同名不同参数

C++ 允许定义**多个同名函数**，只要参数**数量或类型不同**：

```cpp
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

int add(int a, int b, int c) {
    return a + b + c;
}

int main() {
    cout << add(3, 5) << endl;         // 调用 int add(int, int)
    cout << add(3.5, 2.7) << endl;     // 调用 double add(double, double)
    cout << add(1, 2, 3) << endl;      // 调用 int add(int, int, int)
    return 0;
}
```

> 🌟 **好处**：同一个功能的名字，处理不同类型/数量的数据，不用费心想不同名字。

## 6. 函数的嵌套调用 📞

函数里可以调用另一个函数：

```cpp
int square(int n) {         // 求平方
    return n * n;
}

int sumOfSquares(int a, int b) {    // 求两个数的平方和
    return square(a) + square(b);   // 调用 square
}

int main() {
    cout << sumOfSquares(3, 4) << endl;  // 3² + 4² = 25
    return 0;
}
```
