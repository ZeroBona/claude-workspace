# 运算符与表达式 — 讲解 📖

---

## 1. 什么是运算符和表达式？🤔

### 回想数学课

你在数学课上学过：
```
3 + 5 = 8
10 - 3 = 7
4 × 6 = 24
```

在 C++ 里，这些就是**表达式**：
- `3 + 5` → 一个表达式，结果是 8
- `+` 叫**运算符**（告诉计算机做什么运算）
- `3` 和 `5` 叫**操作数**（参与运算的对象）

> 🔑 **表达式** = 由运算符和操作数组成的"算式"

## 2. 算术运算符 🔢

### 基本算术运算符

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加 | `10 + 3` | 13 |
| `-` | 减 | `10 - 3` | 7 |
| `*` | 乘 | `10 * 3` | 30 |
| `/` | 除 | `10 / 3` | 3 ⚠️ |
| `%` | 取余（模） | `10 % 3` | 1 |

> ⚠️ **特别注意——整数除法**：
> - `10 / 3` 的结果是 **3**，不是 3.333！
> - 因为 **int ÷ int 结果还是 int**，小数部分被**直接丢掉**
> - 想要小数结果 → 至少一边是 `double`：`10.0 / 3` 或 `(double)10 / 3`

### 取余运算 %（超有用！）

`%` 求的是**除法余数**：

```cpp
10 % 3 = 1    // 10 ÷ 3 = 3 余 1
15 % 5 = 0    // 15 ÷ 5 = 3 余 0（能整除！）
7 % 9 = 7     // 7 ÷ 9 = 0 余 7
```

> 🌟 **% 的妙用**：
> - `x % 2 == 0` → x 是偶数
> - `x % 2 == 1` → x 是奇数
> - 取个位：`1234 % 10 = 4`

### 运算顺序（和数学一样）

```
1. 先算括号： ( )
2. 再算乘除： * / %
3. 最后算加减： + -
```

```cpp
int result = 2 + 3 * 4;     // 结果 = 2 + 12 = 14
int result2 = (2 + 3) * 4;  // 结果 = 5 * 4 = 20
```

## 3. 自增和自减 ++ / -- ⬆️⬇️

### 让变量加1或减1

```cpp
int count = 5;

count++;   // 等价于 count = count + 1，现在 count = 6
count--;   // 等价于 count = count - 1，现在 count = 5
```

### 前置 vs 后置

```cpp
int a = 5;
int b = a++;   // 后置++：先赋值，再加1
               // b = 5, a = 6

int c = 5;
int d = ++c;   // 前置++：先加1，再赋值
               // c = 6, d = 6
```

> 💡 **初学者建议**：单独用 `++` 和 `--`（不要放在表达式里），就不需要纠结前后了：
> ```cpp
> count++;        // ✅ 这样用，简单明了
> int x = a++;    // ⚠️ 容易混淆，先避免
> ```

## 4. 赋值运算符 📝

### 基本赋值 `=`

```cpp
int x = 10;      // 把 10 赋值给 x
x = x + 5;       // 先算右边 x+5=15，再把 15 赋值给 x
```

### 复合赋值运算符

| 运算符 | 示例 | 等价于 |
|--------|------|--------|
| `+=` | `a += 5` | `a = a + 5` |
| `-=` | `a -= 3` | `a = a - 3` |
| `*=` | `a *= 2` | `a = a * 2` |
| `/=` | `a /= 4` | `a = a / 4` |
| `%=` | `a %= 3` | `a = a % 3` |

```cpp
int score = 100;
score += 10;      // score = 110  ✅ 比 score = score + 10 更简洁
score -= 20;      // score = 90
score *= 2;       // score = 180
```

> 🌟 **复合赋值**：写起来更简洁，读起来更"自然"——"score 加 10"

## 5. 比较运算符 ⚖️

### 用来判断"对还是错"

比较的结果是 `bool` 类型：`true`（真）或 `false`（假）

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 5` | `true` |
| `!=` | 不等于 | `5 != 3` | `true` |
| `<` | 小于 | `3 < 5` | `true` |
| `>` | 大于 | `3 > 5` | `false` |
| `<=` | 小于等于 | `5 <= 5` | `true` |
| `>=` | 大于等于 | `6 >= 5` | `true` |

```cpp
int age = 12;
bool canVote = age >= 18;        // false，12 >= 18 不成立
bool isTeen = age >= 10 && age < 20;   // true
```

> ⚠️ **新手最容易犯的错**：
> - `=` 是赋值！`x = 5` 是把 5 给 x
> - `==` 才是判断相等！`x == 5` 是问"x 等于 5 吗？"

## 6. 逻辑运算符 🧩

### 用来组合多个条件

| 运算符 | 含义 | 示例 | 说明 |
|--------|------|------|------|
| `&&` | 与（并且） | `条件1 && 条件2` | **两个都**成立才为真 |
| `\|\|` | 或（或者） | `条件1 \|\| 条件2` | **一个成立**就为真 |
| `!` | 非（取反） | `!条件` | 真变假，假变真 |

### &&（与）— 两个都要满足

```cpp
int score = 85;
bool pass = score >= 60;              // true
bool excellent = score >= 90;         // false
bool passAndExcellent = score >= 60 && score >= 90;  // false（两个都要满足）
```

```
真 && 真 → 真
真 && 假 → 假
假 && 真 → 假
假 && 假 → 假
```

### ||（或）— 满足一个就行

```cpp
bool isWeekend = true;
bool isHoliday = false;
bool restDay = isWeekend || isHoliday;   // true（有一个成立就行）
```

```
真 || 真 → 真
真 || 假 → 真
假 || 真 → 真
假 || 假 → 假
```

### !（非）— 反过来

```cpp
bool isRainy = false;
bool isSunny = !isRainy;     // true（非假即真）
```

### 综合示例

```cpp
int age = 13;
bool hasTicket = true;

// 可以进游乐园吗？(年龄≥10 并且 有票) 或 年龄<10 但免费
bool canEnter = (age >= 10 && hasTicket) || age < 3;

cout << canEnter << endl;   // 输出 1（true）
```

## 7. 运算符优先级总表 📊

### 从高到低（常用部分）

| 优先级 | 运算符 | 说明 |
|--------|--------|------|
| 1（最高） | `()` | 括号 |
| 2 | `++` `--` | 自增自减 |
| 3 | `!` | 逻辑非 |
| 4 | `*` `/` `%` | 乘除取余 |
| 5 | `+` `-` | 加减 |
| 6 | `<` `<=` `>` `>=` | 比较 |
| 7 | `==` `!=` | 等于不等于 |
| 8 | `&&` | 逻辑与 |
| 9 | `\|\|` | 逻辑或 |
| 10 | `=` `+=` `-=` 等 | 赋值 |

### 记不住怎么办？🤯

> 🌟 **黄金法则：拿不准就加括号！**
> ```cpp
> // 这样写——不用记优先级
> bool result = ((a + b) > c) && (d != e);
> ```
