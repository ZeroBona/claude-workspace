# 练习题 📝

## ⭐ 基础题

### 第1题：算术运算练习
编写程序，定义 `a = 25, b = 7`，输出以下结果：
```
25 + 7 = 32
25 - 7 = 18
25 * 7 = 175
25 / 7 = 3
25 % 7 = 4
```

### 第2题：自增自减
猜一猜下面程序输出什么？然后写代码验证。

```cpp
int x = 3;
int y = x++;
int z = ++x;
cout << x << " " << y << " " << z << endl;
```

### 第3题：判断奇偶
编写程序，输入一个整数，输出它是奇数还是偶数。

**运行效果**：
```
请输入一个整数：42
42是偶数
```

## ⭐⭐⭐ 挑战题

### 第7题：抽奖条件判定 🎰
编写程序，输入一个整数号码，判断是否中奖。

中奖规则（满足**任意一条**即可）：
1. 号码能被 7 整除
2. 号码的个位数是 8
3. 号码 > 100 且是偶数

**运行效果**：
```
请输入号码：28
🎉 中奖了！（能被7整除）

请输入号码：58
🎉 中奖了！（个位是8）

请输入号码：102
🎉 中奖了！（大于100且是偶数）

请输入号码：33
😅 没中奖
```

> **提示**：用 `||`（逻辑或）组合三个条件

### 第8题：一元二次方程判别式 📐
输入三个整数 a, b, c，计算一元二次方程 `ax² + bx + c = 0` 的判别式 `Δ = b² - 4ac`，并判断：

- 如果 Δ > 0，输出"有两个不同的实数根"
- 如果 Δ = 0，输出"有两个相同的实数根"
- 如果 Δ < 0，输出"没有实数根"

**运行效果**：
```
请输入 a b c：1 -5 6
Δ = 1，有两个不同的实数根
```

### 第7题
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "请输入号码：";
    cin >> n;

    bool cond1 = n % 7 == 0;
    bool cond2 = n % 10 == 8;
    bool cond3 = n > 100 && n % 2 == 0;

    if (cond1 || cond2 || cond3) {
        cout << "🎉 中奖了！";
        if (cond1) cout << "（能被7整除）";
        else if (cond2) cout << "（个位是8）";
        else cout << "（大于100且是偶数）";
        cout << endl;
    } else {
        cout << "😅 没中奖" << endl;
    }

    return 0;
}
```
