# 指针与引用 — 讲解 📖

---

## 1. 内存地址——变量的"门牌号" 🏠

### 变量在内存中的位置

每个变量都存储在内存中，有**自己的地址**：

```cpp
int x = 10;
cout << &x << endl;   // 输出 x 的地址，如：0x61fe14
```

`&` 是"取地址符"——拿到变量的**门牌号**。

### 把内存想象成一排带编号的邮箱

```
内存：
┌────┬────┬────┬────┬────┬────┬────┐
│    │    │ 10 │    │    │    │    │
└─┬──┴─┬──┴─┬──┴─┬──┴─┬──┴─┬──┴─┬──┘
 1000 1004 1008 1012 1016 1020 1024  ← 地址（举例）

变量 x 在地址 1008，存储的值是 10
```

## 2. 指针——存地址的变量 📍

### 什么是指针？

**指针**是一个特殊的变量——它存的是**另一个变量的地址**。

```cpp
int x = 10;       // 普通变量：存值
int *p = &x;      // 指针变量：存 x 的地址
```

```
变量 x：
  ┌────┐
  │ 10 │    ← 值
  └────┘
  地址：0x1008

指针 p：
  ┌──────────┐
  │ 0x1008   │    ← 存的是 x 的地址
  └──────────┘
  地址：0x2000
```

### 定义指针

```cpp
类型 *指针名;

int *p;        // p 是一个指向 int 的指针
double *q;     // q 是一个指向 double 的指针
string *s;     // s 是一个指向 string 的指针
```

### 两个重要运算符

| 运算符 | 名称 | 作用 |
|--------|------|------|
| `&` | 取地址 | 获取变量的地址 |
| `*` | 解引用 | 通过地址访问变量 |

```cpp
int x = 10;
int *p = &x;     // p 指向 x（p 存了 x 的地址）

cout << p << endl;   // 输出 p 的值（也就是 x 的地址）
cout << *p << endl;  // 输出 p 指向的值（也就是 x 的值 10）

*p = 20;             // 通过指针修改 x 的值
cout << x << endl;   // 20！x 被修改了
```

> 💡 **比喻**：
> - `p` = 一张纸条上写了 x 的门牌号
> - `*p` = 顺着门牌号找到 x 的家，看里面有什么

## 3. 指针和函数 — 用指针修改外部变量 🔧

```cpp
void swap(int *a, int *b) {   // 接收指针
    int temp = *a;            // *a 取 a 指向的值
    *a = *b;
    *b = temp;
}

int main() {
    int x = 10, y = 20;
    swap(&x, &y);             // 传地址
    cout << x << " " << y << endl;  // 20 10
    return 0;
}
```

### 三种方式的对比

| 方式 | 参数写法 | 调用写法 | 能改原变量？ |
|------|---------|---------|------------|
| 值传递 | `int a` | `func(x)` | ❌ |
| 引用传递 | `int &a` | `func(x)` | ✅ |
| 指针传递 | `int *a` | `func(&x)` | ✅ |

## 4. 引用——变量的"别名" 🔗

### 引用就是给变量起个小名

```cpp
int x = 10;
int &ref = x;      // ref 是 x 的引用（别名）

ref = 20;          // 修改 ref 就是修改 x
cout << x << endl; // 20
```

> 🌟 **引用 vs 指针**：
> - 引用：给变量起别名，用起来和原变量一样
> - 指针：存变量的地址，需要用 `*` 解引用
> - **引用更安全**，推荐在函数参数中多用引用

```cpp
// 引用做参数（推荐）
void swap(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}

int x = 10, y = 20;
swap(x, y);   // 不用写 &，用起来像值传递，但能改原变量
```

## 5. 指针的常见用途 🛠️

### 遍历数组

数组名就是数组首元素的地址：

```cpp
int arr[] = {10, 20, 30, 40};
int *p = arr;           // arr 就是 &arr[0]

cout << *p << endl;     // 10（第一个元素）
p++;                    // 指向下一个位置
cout << *p << endl;     // 20（第二个元素）
```

### 动态内存（了解）

```cpp
int *p = new int;       // 在堆上分配一个 int
*p = 42;
cout << *p << endl;
delete p;               // 用完记得释放

int *arr = new int[10]; // 动态数组
// ... 使用数组 ...
delete[] arr;           // 释放动态数组
```

> ⚠️ **初学提示**：指针是 C++ 中最容易出错的部分。
> - 定义指针后**一定要初始化**，否则叫"野指针"
> - 用 `new` 分配了内存**一定要用 delete 释放**，否则"内存泄漏"
