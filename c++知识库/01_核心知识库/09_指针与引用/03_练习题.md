# 练习题 📝

## ⭐ 基础题

### 第1题：指针基础
编写程序，定义一个 int 变量和一个指向它的指针，分别用变量名和指针修改它的值。

### 第2题：用指针求数组最大值
编写函数 `int findMax(int *arr, int n)`，用指针方式遍历数组找最大值。

## ⭐⭐ 进阶题

### 第3题：用指针反转数组
编写函数 `void reverse(int *arr, int n)` 用指针反转数组。

## ⭐⭐⭐ 挑战题

### 第4题：字符串长度（指针版）
编写函数 `int strLength(const char *s)`，用指针计算字符串长度（不能用下标）。
