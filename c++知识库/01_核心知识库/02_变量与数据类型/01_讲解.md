# 变量与数据类型 — 讲解 📖

---

## 1. 什么是变量？📦

### 生活中的比喻

想象你的书桌上放了几个盒子：

```
┌──────────────────┐
│   盒子 A          │
│   值：12          │  ← 这个盒子装的是数字 12
│   标签：年龄       │
└──────────────────┘

┌──────────────────┐
│   盒子 B          │
│   值："小明"       │  ← 这个盒子装的是文字"小明"
│   标签：姓名       │
└──────────────────┘

┌──────────────────┐
│   盒子 C          │
│   值：3.14        │  ← 这个盒子装的是小数 3.14
│   标签：圆周率     │
└──────────────────┘
```

**变量**就是计算机内存里的"盒子"：
- 每个盒子有**名字**（标签）
- 每个盒子能**装东西**（数据）
- 盒子里的东西可以**换**（变量的值可以变）

### 变量 = 内存中的"带名字的存储空间"

```cpp
int age;           // 告诉计算机：给我一个叫 age 的盒子，用来装整数
age = 12;          // 在 age 盒子里放上数字 12
cout << age;       // 看看盒子里装了什么 → 输出 12
age = 13;          // 换个东西放进去！现在 age 变成 13 了
```

### 定义变量的语法

```cpp
数据类型 变量名;               // 只定义，不放值
数据类型 变量名 = 初始值;      // 定义的同时放一个值进去
```

> 💡 **专业术语**：
> - `int age;` → 叫"**变量定义**"
> - `age = 12;` → 叫"**赋值**"
> - `int age = 12;` → 叫"**初始化**"（定义 + 第一次赋值）

## 2. 变量的命名规则 🏷️

### ✅ 可以用的字符

| 字符 | 示例 | 说明 |
|------|------|------|
| 英文字母 | `age`, `myName` | 大小写字母都可以 |
| 数字 | `num1`, `score2` | **不能开头** |
| 下划线 | `_count`, `my_age` | 常用于长单词分隔 |

### ✅ 好名字 vs ❌ 坏名字

| ✅ 好名字 | ❌ 坏名字 | 为什么？ |
|-----------|-----------|---------|
| `age` | `a` | a 看不出是什么 |
| `studentScore` | `ss` | ss 让人看不懂 |
| `myName` | `我的名字` | C++ 不能用中文变量名！ |
| `totalCount` | `2count` | 数字不能开头 |
| `my_age` | `my age` | 不能有空格 |

### 命名风格推荐（驼峰命名法 🐫）

```
小驼峰（推荐）：myName, studentScore, totalCount
大驼峰：MyName, StudentScore, TotalCount
下划线式：my_name, student_score, total_count
```

> 🌟 **初一阶段推荐**：用英文单词或拼音，让人一看就懂！
> - 年龄 → `age`
> - 姓名 → `name`
> - 分数 → `score`
> - 总分 → `total`

### 不能用的名字（保留字/关键词）

这些词已经被 C++ 自己占用了，不能当变量名：

```
int, double, char, bool, if, else, for, while,
return, main, true, false, using, namespace ...
```

> 💡 Dev-C++ 里这些词会**变蓝色**，看到蓝色的词就别当变量名了！

## 3. 五种基本数据类型 🔢

### 概览表

| 数据类型 | 关键词 | 能装什么 | 举例 | 占多大 |
|----------|--------|---------|------|--------|
| **整型** | `int` | 整数 | `-100, 0, 99` | 4 字节 |
| **浮点型** | `double` | 小数（双精度） | `3.14, -0.5` | 8 字节 |
| **字符型** | `char` | 单个字符 | `'A', '9', '$'` | 1 字节 |
| **布尔型** | `bool` | 真/假 | `true, false` | 1 字节 |
| **字符串** | `string` | 一串文字 | `"你好"`, `"C++"` | 不定 |

> 🌟 **最常用**：`int` 和 `double` 和 `string`

### 3.1 int — 整数

```cpp
int age = 12;
int score = 95;
int temperature = -5;
```

整数的范围：大约 **-21亿 ~ 21亿**（对初学者来说完全够用）

### 3.2 double — 小数（浮点数）

```cpp
double pi = 3.14159;
double price = 19.9;
double height = 1.65;
```

> `double` 比 `float`（另一种小数类型）精度更高，**初学用 double 就够了**

### 3.3 char — 单个字符

```cpp
char grade = 'A';
char symbol = '#';
char digit = '5';
```

> ⚠️ **注意**：`char` 用**单引号** `' '`，不是双引号！
> - ✅ `char c = 'A';`
> - ❌ `char c = "A";`

### 3.4 bool — 布尔值（真假）

```cpp
bool isPass = true;     // 及格了
bool isRainy = false;   // 没下雨
```

`bool` 只有两个值：
- `true`（真，也可以理解为 1）
- `false`（假，也可以理解为 0）

### 3.5 string — 字符串

```cpp
string name = "小明";
string greeting = "你好，世界！";
```

> ⚠️ **注意**：`string` 用**双引号** `" "`，且需要 `#include <string>` 或在 `#include <iostream>` 下也能用

## 4. 变量的使用与计算 📐

### 定义多个变量

```cpp
int a, b, c;                // 一次定义三个 int 变量
double x = 1.5, y = 2.5;   // 定义两个 double 并初始化
```

### 变量的赋值与更新

```cpp
int count = 10;    // 定义 count 并赋值为 10
count = 20;        // 把 count 改成 20（旧值 10 被覆盖）
count = count + 5; // count 变成 25 —— 先取出 count 的值 20，加 5，再放回去！
```

> 💡 **思考题**：为什么 `count = count + 5;` 两边都有 count？
> 计算机先看**右边**：取出 count 的值（20），加上 5 → 得到 25
> 然后把这个结果**赋值给左边**的 count → count 变成 25

### 不同类型混用

```cpp
int a = 5;
double b = 2.5;
double c = a + b;   // 结果是 7.5（int 自动转成 double 再算）
```

## 5. 常量 🔒

### 什么是常量？

常量就像**封了口的盒子**——放了东西进去就**不能改了**。

### 用 const 定义常量

```cpp
const double PI = 3.14159;   // PI 是常量，不能修改
const int DAYS_IN_WEEK = 7;

PI = 3.14;  // ❌ 错误！常量不能改！
```

### 什么时候用常量？

- 圆周率、固定倍数、税率等**不会变的值**
- 用常量能让代码更清晰，也防止不小心改错

> 🌟 **命名习惯**：常量名一般**全大写**，如 `PI`、`MAX_SCORE`

## 6. 类型转换 🎭

### 自动转换（隐式转换）

```cpp
int a = 10;
double b = a;       // int 自动变成 double → b = 10.0
```

### 强制转换（显式转换）

```cpp
double pi = 3.14159;
int x = (int)pi;    // 强制把小数转成整数 → x = 3（去掉小数部分！）
```

> ⚠️ **注意**：强制转换会**丢失精度**！`3.14159` 转成 `int` 就变成了 `3`。
