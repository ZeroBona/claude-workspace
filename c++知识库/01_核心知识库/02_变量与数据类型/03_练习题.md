# 练习题 📝

## ⭐ 基础题

### 第1题：修改变量输出
编写程序，定义以下变量并输出：

- 你最喜欢的数字（整数）
- 你的身高（小数）
- 你名字的首字母（字符）
- 今天是晴天吗？（是/否用布尔值表示）

### 第2题：变量运算
定义两个整数 `a = 15` 和 `b = 4`，输出：

```
a + b = 19
a - b = 11
a * b = 60
a / b = 3
```

> **思考**：为什么 15 ÷ 4 的结果是 3，而不是 3.75？

### 第3题：温度转换器
编写程序，输入一个摄氏温度 `c`，输出对应的华氏温度 `f`。

> **公式**：`f = c * 9 / 5 + 32`

## ⭐⭐ 进阶题

### 第4题：平均分计算器 🏆
编写程序，输入三科成绩（语文、数学、英语），计算总分和平均分。

**运行效果示例**：
```
=== 成绩计算器 ===
请输入语文成绩：88
请输入数学成绩：95
请输入英语成绩：90

总分 = 273
平均分 = 91
```

### 第5题：矩形信息计算器 📐
编写程序，输入矩形的长和宽，计算周长和面积。

> **公式**：周长 = 2 × (长 + 宽)，面积 = 长 × 宽

**运行效果示例**：
```
=== 矩形计算器 ===
请输入长：8
请输入宽：5

周长 = 26
面积 = 40
```

---

## ⭐⭐⭐ 挑战题

### 第6题：时间转换器 ⏰
编写程序，输入总共的分钟数，把它转换成"几小时几分钟"。

**运行效果示例**：
```
=== 时间转换器 ===
请输入分钟数：135
135分钟 = 2小时15分钟
```

> **提示**：用 `/` 求商，用 `%` 求余数

### 第7题：数字拆分器 🔢
编写程序，输入一个四位数，分别输出它的个位、十位、百位、千位。

**运行效果示例**：
```
=== 数字拆解 ===
请输入一个四位数：3782

千位：3
百位：7
十位：8
个位：2
```

> **提示**：用 `% 10` 取个位，用 `/ 10` 去掉个位

#include <iostream>

    double c, f;

### 第6题
```cpp
#include <iostream>
using namespace std;

int main() {
    int totalMinutes;
    cout << "=== 时间转换器 ===" << endl;
    cout << "请输入分钟数：";
    cin >> totalMinutes;

    int hours = totalMinutes / 60;
    int minutes = totalMinutes % 60;

    cout << totalMinutes << "分钟 = " << hours << "小时" << minutes << "分钟" << endl;
    return 0;
}
```

### 第7题

#include <iostream>
using namespace std;

int main() {
    int num;
    cout << "=== 数字拆解 ===" << endl;
    cout << "请输入一个四位数：";
    cin >> num;

    int ge = num % 10;
    int shi = num / 10 % 10;
    int bai = num / 100 % 10;
    int qian = num / 1000;

    cout << "千位：" << qian << endl;
    cout << "百位：" << bai << endl;
    cout << "十位：" << shi << endl;
    cout << "个位：" << ge << endl;
    return 0;
}
