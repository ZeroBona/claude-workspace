# 项目二：猜单词游戏 🎮

---

## 项目要求

开发一个猜单词游戏，规则如下：

1. 程序从单词列表中随机选一个单词
2. 每次显示单词的字母进度（猜对的显示字母，没猜的显示 _ ）
3. 玩家每次猜一个字母
4. 猜对了就显示该字母的位置
5. 猜错了记录错误次数
6. 全部猜完或错误达到上限则游戏结束

## 涉及的知识点

| 知识点 | 用在哪儿 |
|-------|---------|
| 数组/vector | 存单词列表 |
| string | 单词和进度显示 |
| 循环 | 游戏主循环 |
| 随机数 | 随机选单词 |
| 函数 | 模块化 |

---

## 参考代码

**文件**：[code/02_word_guess.cpp](code/02_word_guess.cpp)

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <cstdlib>
#include <ctime>
using namespace std;

// 单词列表
const vector<string> WORDS = {
    "computer", "program", "language", "science",
    "algorithm", "keyboard", "monitor", "software"
};

string chooseWord() {
    srand(time(0));
    return WORDS[rand() % WORDS.size()];
}

string initProgress(const string &word) {
    return string(word.length(), '_');
}

void showProgress(const string &progress, int wrongCount) {
    for (char c : progress) cout << c << " ";
    cout << "\n错误次数：" << wrongCount << endl;
}

bool isGuessRight(const string &word, char guess, string &progress) {
    bool found = false;
    for (size_t i = 0; i < word.length(); i++) {
        if (word[i] == guess) {
            progress[i] = guess;
            found = true;
        }
    }
    return found;
}

int main() {
    const int MAX_WRONG = 6;
    string word = chooseWord();
    string progress = initProgress(word);
    int wrongCount = 0;
    vector<char> guessedLetters;

    cout << "=== 猜单词游戏 ===" << endl;
    cout << "单词长度：" << word.length() << "，最多猜错" << MAX_WRONG << "次" << endl;

    while (wrongCount < MAX_WRONG && progress != word) {
        cout << "\n当前进度：";
        showProgress(progress, wrongCount);

        char guess;
        cout << "猜一个字母：";
        cin >> guess;

        // 检查是否已猜过
        bool alreadyGuessed = false;
        for (char c : guessedLetters) {
            if (c == guess) {
                alreadyGuessed = true;
                break;
            }
        }

        if (alreadyGuessed) {
            cout << "这个字母已经猜过了！" << endl;
            continue;
        }

        guessedLetters.push_back(guess);

        if (isGuessRight(word, guess, progress)) {
            cout << "✅ 猜对了！" << endl;
        } else {
            cout << "❌ 猜错了！" << endl;
            wrongCount++;
        }
    }

    cout << "\n=== 游戏结束 ===" << endl;
    if (progress == word) {
        cout << "🎉 恭喜你猜对了！单词是：" << word << endl;
    } else {
        cout << "😅 游戏结束，正确答案是：" << word << endl;
    }

    return 0;
}
```
