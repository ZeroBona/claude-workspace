# C++ 与信息学奥赛 🏅

> 学 C++ 不仅能做游戏、写软件，还能参加竞赛，拿奖升学！

---

## 🤔 什么是信息学奥赛？

全称：**全国青少年信息学奥林匹克竞赛**（简称 NOI）

和数学奥赛、物理奥赛一样，是教育部认可的五大奥赛之一。比赛的**指定语言就是 C++**。

### 竞赛阶梯

```
入门级                 进阶级                   顶配级
CSP-J/S（省级） →  NOIP（省级联赛） →  NOI（全国赛） →  IOI（世界赛）
   初中生可参加        高中生                 国家队              全球顶尖
```

---

## 🏆 为什么参加信息学奥赛？

### 1. 升学优势
- CSP-J/S 获奖 → 重点高中自主招生优势
- NOIP/NOI 获奖 → 保送清北等名校
- 很多省份中考有科技特长生政策

### 2. 能力提升
- 🤔 **逻辑思维**：写代码就是训练逻辑
- 🧩 **解决问题**：把大问题拆成小问题
- 🔍 **严谨细心**：一个符号错了都跑不通
- 💪 **抗压能力**：比赛时能保持冷静

### 3. 未来发展
- 大学计算机专业优势巨大
- 大公司（腾讯、阿里、字节）招聘看重算法能力
- 人工智能时代，编程是核心技能

## 📚 竞赛主要考察什么？

| 知识点 | 对应单元 | 难度 |
|--------|---------|------|
| 顺序结构、输入输出 | 01 | ⭐ |
| 分支结构 | 04 | ⭐ |
| 循环结构 | 05 | ⭐ |
| 数组、字符串 | 06-07 | ⭐⭐ |
| 函数 | 08 | ⭐⭐ |
| 结构体 | 10 | ⭐⭐ |
| 排序、查找 | 06+13 | ⭐⭐ |
| 栈、队列 | 13 | ⭐⭐⭐ |
| 递归、搜索 | 进阶 | ⭐⭐⭐ |
| 动态规划、图论 | 进阶 | ⭐⭐⭐⭐⭐ |

> 💡 这套知识库的 01~13 单元，基本覆盖了 CSP-J（入门组）的全部考点！

## 🎯 竞赛路线图（初中生）

### 初一（现在！）
- ✅ 学完 01~07 单元（基础语法）
- ✅ 多在 [洛谷](https://www.luogu.com.cn) 刷题
- ✅ 尝试参加 CSP-J 第一轮（试试水）

### 初二
- ✅ 学完 08~13 单元（函数、结构体、STL）
- ✅ 学习基础算法（递归、搜索）
- ✅ CSP-J 争取二等奖以上

### 初三
- ✅ 学习进阶算法（DP、图论）
- ✅ CSP-J 一等奖 / 尝试 CSP-S
- ✅ 中考科技特长生
