# C++ 发展简史 📜

> 从"带类的C"到世界上最强大的编程语言之一

---

## 🎂 1979年 — C++ 的诞生

1979年，一位叫 **本贾尼·斯特劳斯特卢普（Bjarne Stroustrup）** 的丹麦计算机科学家在贝尔实验室工作。

他当时遇到的问题：想写一个程序模拟分布式系统，但现有的语言要么太慢、要么太难用。

于是他做了一个大胆的决定——**自己创造一种新语言**！

他把 C 语言（当时最流行的系统编程语言）和他喜欢的 Simula 语言（有"类"的概念）结合起来，最初取名叫 **"带类的 C"（C with Classes）**。

> 💡 **那年他才 29 岁**，和我们一样只是想做点有趣的东西，没想到创造了一个改变世界的语言！

---

## 🚀 1983年 — 正式命名为 C++

1983年，这种语言正式改名为 **C++**。

名字来源很有编程特色：
- `C` 表示它继承了 C 语言
- `++` 是 C 语言中的"自增运算符"（加1）
- 合起来就是 **"C 语言的升级版"**

> 这个命名来自斯特劳斯特卢普的同事 Rick Mascitti，他觉得"带类的C"太难听了！

---

## 📗 1985年 — 第一本《C++程序设计语言》

斯特劳斯特卢普写了第一本 C++ 教材，他自己一个人写了第一个 C++ 编译器（Cfront）。

同年，第一个**商业版 C++** 发布。

---

## 🏛️ 1998年 — C++ 第一个国际标准

1998年，C++ 正式成为国际标准（称为 **C++98**）。

这意味着：
- 不同厂商的编译器遵循统一的规则
- 写一次代码，可以在不同电脑上编译运行
- C++ 正式成为"正规军"！

---

## 🔄 后续版本

| 年份 | 标准名称 | 重要新特性 |
|------|---------|-----------|
| 1998 | C++98 | 第一个标准版本 |
| 2005 | C++03 | 小修小补 |
| **2011** | **C++11** | **重大更新！** auto、智能指针、Lambda |
| 2014 | C++14 | 小优化 |
| **2017** | **C++17** | 文件系统、并行算法 |
| **2020** | **C++20** | 模块、协程（超大更新） |
| 2023 | C++23 | 最新标准 |

> 🌟 **C++11 是里程碑**，距离 C++98 过了 13 年，加入了大量新特性。现在学 C++ 其实学的是 C++11 之后的新标准！

---

## 🎮 C++ 今天在做什么？

C++ 仍然是世界上最快的编程语言之一，它活跃在这些领域：

- 🎮 **游戏开发**：虚幻引擎、Unity（部分）都用 C++
- 🌐 **浏览器**：Chrome、Edge 的核心是 C++
- 🏎️ **自动驾驶**：很多汽车系统用 C++
- 🚀 **航天**：NASA 的火星探测器用 C++
- 🎬 **电影特效**：《阿凡达》《复仇者联盟》的特效软件用 C++
- 💻 **信息学奥赛（NOI）**：指定使用 C++
