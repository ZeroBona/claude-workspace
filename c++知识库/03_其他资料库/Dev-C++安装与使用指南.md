# Dev-C++ 安装与使用指南 🛠️

> Dev-C++ 是一款轻量级的 C++ 编译器，适合新手入门，很多信息学奥赛选手也用。

---

## 📥 下载与安装

### 方式一：直接下载（推荐）
搜索引擎搜 **"Dev-C++ 下载"**，选择 Orwell Dev-C++ 版本（最新的维护版本）。

### 方式二：官网下载
1. 访问 https://sourceforge.net/projects/orwelldevcpp/
2. 点击 Download 按钮
3. 下载完成后双击安装
4. 一路点"下一步"(Next)即可

### 安装注意事项
- 语言选择：简体中文 / English
- 安装路径：默认即可
- 组件：全部勾上

---

## 🚀 基本使用

### 1. 新建文件
```
菜单 → 文件 → 新建 → 源代码
或 快捷键 Ctrl + N
```

### 2. 写代码
在白色区域输入代码：
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

### 3. 保存文件
```
菜单 → 文件 → 保存
或 快捷键 Ctrl + S
```
> 💡 文件名用英文，如 `hello.cpp`，不要用中文！

### 4. 编译运行
```
菜单 → 运行 → 编译运行
或 点击工具栏的 ▶ 按钮
或 快捷键 F11
```

### 5. 只看编译（不运行）
```
菜单 → 运行 → 编译
或 快捷键 Ctrl + F9
```
> 如果只想检查有没有语法错误，用这个。

## ⚙️ 常用设置

### 设置字体大小
```
工具 → 编译器选项 → 显示 → 编辑器字体 → 选择大小（推荐 14~16）
```

### 设置 C++11 标准（重要！）
```
工具 → 编译选项 → 编译器 → 
勾选「编译时加入以下命令」→ 
输入：-std=c++11
```
> 如果不设置，有些新语法（如范围 for 循环）会报错！

### 显示行号
```
工具 → 编辑器选项 → 显示 → 勾选「显示行号」
```

## 🐛 常见问题

### Q：运行后窗口一闪而过？
**原因**：程序运行完自动关闭了。

**解决方法**：在 `return 0;` 前加一行：
```cpp
system("pause");
```
或添加：
```cpp
cin.get();  // 等待按回车
```

### Q：编译后中文乱码？
**原因**：编码不一致。

**解决方法**：
```
工具 → 编译选项 → 编译器 →
在输入框中加入：-fexec-charset=GBK
```

### Q：按钮都是灰色的点不了？
**原因**：没有打开任何文件。先新建或打开一个 .cpp 文件。

## 💡 小贴士

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 编译运行 | `F11` | 最常用的 |
| 仅编译 | `Ctrl+F9` | 只检查语法 |
| 新建文件 | `Ctrl+N` | 新建一个 .cpp |
| 保存 | `Ctrl+S` | 经常按！ |
| 撤销 | `Ctrl+Z` | 写错了就按 |
| 全选 | `Ctrl+A` | 选中全部代码 |

> 🌟 **记住**：每次写新代码前，先按 `Ctrl+N` 新建文件，不要直接在旧代码上改！
