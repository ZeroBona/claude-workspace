# C++ 运算符优先级速查表 📊

> 优先级数字越小，越先计算。拿不准就加括号 `()`！

---

## 📋 完整优先级表（从高到低）

| 优先级 | 运算符 | 名称 | 结合性 |
|--------|--------|------|--------|
| 1 | `::` | 作用域解析 | 左→右 |
| 2 | `()` `[]` `->` `.` | 函数调用、下标、成员访问 | 左→右 |
| 3 | `++` `--`（后置） | 后置自增/自减 | 左→右 |
| 4 | `++` `--`（前置） | 前置自增/自减 | 右→左 |
| 4 | `!` `~` | 逻辑非、按位取反 | 右→左 |
| 4 | `+` `-`（一元） | 正号、负号 | 右→左 |
| 4 | `*` `&` | 解引用、取地址 | 右→左 |
| 4 | `(类型)` | 强制类型转换 | 右→左 |
| 5 | `*` `/` `%` | 乘、除、取余 | 左→右 |
| 6 | `+` `-`（二元） | 加、减 | 左→右 |
| 7 | `<<` `>>` | 左移、右移 / 输出、输入 | 左→右 |
| 8 | `<` `<=` `>` `>=` | 关系比较 | 左→右 |
| 9 | `==` `!=` | 等于、不等于 | 左→右 |
| 10 | `&&`（按位与） | | 左→右 |
| 11 | `^`（按位异或） | | 左→右 |
| 12 | `|`（按位或） | | 左→右 |
| 13 | `&&` | 逻辑与（AND） | 左→右 |
| 14 | `||` | 逻辑或（OR） | 左→右 |
| 15 | `?:` | 三元条件运算符 | 右→左 |
| 16 | `=` `+=` `-=` `*=` `/=` `%=` | 赋值、复合赋值 | 右→左 |
| 17 | `,` | 逗号运算符 | 左→右 |

## 🎯 初学重点（灰色=不常用）

```
高优先级             低优先级
──────────────────────────────→

( )  ++ --  !    * / %    + -    < > == !=    &&    ||    = +=
括号  自增  逻辑非 乘除取余  加减   比较      逻辑与 逻辑或  赋值
```

### 记忆口诀

> **括号自增最先算，乘除加减其次来。**
> **比较等于再逻辑，赋值运算符最末排。**

## ⚠️ 常见陷阱

### 陷阱 1：`==` 和 `=` 混用
```cpp
// ❌ 本意是判断 a 是否等于 5
if (a = 5) { ... }    // 这是赋值！a 变成 5，条件永远为真

// ✅ 正确
if (a == 5) { ... }
```

### 陷阱 2：`&&` 和 `||` 混用
```cpp
// ❌ 本意是 a>3 且 a<10
if (a > 3 && a < 10) { ... }

// ✅ 正确写法
```

### 陷阱 3：自增自减混在表达式里
```cpp
int a = 5;
int b = a++ + ++a;   // ❌ 不同编译器结果不同！不要这样写

// ✅ 分开写
int a = 5;
a++;
int c = a + a;       // 结果明确
```

## 🌟 黄金法则

> **拿不准就加括号！不会影响性能，但能让代码意图更清晰！**

```cpp
// 不用记优先级
bool result = ((a + b) > c) && (d != e);
```
