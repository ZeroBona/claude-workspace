# 编程界的重要人物 👨‍💻

> 这些人用代码改变了世界。他们曾经也是普通人，只是坚持做了自己热爱的事。

---

## 🇩🇰 本贾尼·斯特劳斯特卢普 — C++ 之父

```
┌────────────────────────────────────────┐
│  Bjarne Stroustrup                       │
│  1950年出生 · 丹麦 · 计算机科学家        │
│  "我发明 C++ 是为了解决实际问题，          │
│   不是为了让它变得复杂。"                 │
└────────────────────────────────────────┘
```

**他的故事**：
斯特劳斯特卢普在剑桥大学读博时，发现写模拟程序很麻烦——C 语言不够灵活，Simula 又太慢。于是他决定结合两者优点。

他本来想给新语言取名叫 **"C84"**，但同事建议叫 C++，他觉得又幽默又贴切（C 语言的自增运算）。

> 🌟 **给初一生的启示**：遇到问题，不抱怨，想办法解决——这就是编程精神。

---

## 🇺🇸 丹尼斯·里奇 — C 语言之父

```
┌────────────────────────────────────────┐
│  Dennis Ritchie                          │
│  1941~2011 · 美国 · 计算机科学家         │
│  "C 语言是简单、优雅的工具。"             │
└────────────────────────────────────────┘
```

**他的故事**：
里奇在贝尔实验室创造了 **C 语言**（C++ 的"爸爸"）和 **Unix 操作系统**。

没有里奇，就没有 C 语言；没有 C 语言，就没有 C++；没有 C++，就没有你玩的很多游戏。

> 🤫 **你不知道的事**：Windows、Linux、macOS 的内核都是用 C/C++ 写的。你每次开机，都在用他的作品。

---

## 🇫🇮 林纳斯·托瓦兹 — Linux 之父

```
┌────────────────────────────────────────┐
│  Linus Torvalds                         │
│  1969年出生 · 芬兰 · 程序员             │
│  "我只是为了好玩，做了一个操作系统。"     │
└────────────────────────────────────────┘
```

**他的故事**：
1991年，21岁的芬兰大学生林纳斯觉得自己的电脑操作系统不好用，于是**自己写了一个**——这就是 **Linux**。

他用了 C 语言（后来也用了很多 C++），开发了全世界最流行的开源操作系统内核。

> 🌟 **给初一生的启示**：他 21 岁就做出了改变世界的作品。年龄不是问题，兴趣和坚持才是。

---

## 🇺🇸 肯·汤普森 — Unix 联合创始人

```
┌────────────────────────────────────────┐
│  Ken Thompson                            │
│  1943年出生 · 美国 · 计算机科学家        │
└────────────────────────────────────────┘
```

他和里奇一起创造了 Unix 操作系统。为了玩游戏，他写了 Unix 的第一个版本。

他还发明了 **B 语言**（C 语言的"前身"）。1983年获得 **图灵奖**（计算机界的诺贝尔奖）。

---

## 🇺🇸 比雅尼·斯特劳斯特卢普的导师

斯特劳斯特卢普在剑桥的博士导师 **大卫·惠勒** 提出过一个著名原则：

> **"计算机科学中的任何问题，都可以通过增加一层间接层来解决。"**

这句话影响了很多后来的编程设计。

## 📊 人物关系图

```
丹尼斯·里奇 → 创造了 C 语言
       ↓
斯特劳斯特卢普 → 基于 C 创造了 C++
       ↓
林纳斯·托瓦兹 → 用 C 写了 Linux
       ↓
今天无数程序员 → 用 C++ 写游戏、软件、系统
```

---

## 💡 他们的共同点

1. ✅ **从小就对计算机感兴趣**
2. ✅ **遇到问题，自己写工具解决**
3. ✅ **不追求完美，先做出能用的版本**
4. ✅ **乐于分享，让更多人用上自己的作品**

> 你也在学 C++，说不定下一个编程界的重要人物就是你！🚀
