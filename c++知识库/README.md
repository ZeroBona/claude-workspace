## 💬 直接说中文就能用

每个学习环节都有对应的功能，直接说中文即可调用：

### 学习 — 先看看学什么
| 你说 | 效果 |
|------|------|
| `学习` / `课程概览` / `看看课程` | 展示整个课程全景总览 + 学习路线图 + 当前进度 |
| `学习 03` / `课程 03` / `看看03单元` | 展示指定单元的详细内容 + 学习目标 + 进度 |

> 刚开始不知道学啥？说"**学习**"先看看全貌吧！

### 出题 — 学完一单元就测试
| 你说 | 效果 |
|------|------|
| `出题` / `考试` / `测验` / `考考我` | 自动找下一个没出题的单元，生成试卷 |
| `出题 03` / `考试 03` / `测验 03` | 出第03单元的试题 |

### 批改 — 做完卷子对答案
| 你说 | 效果 |
|------|------|
| `批改` / `改卷` / `对答案` | 询问要批改哪个单元 |
| `批改 02` / `改卷 02` | 批改第02单元的试卷 |
| `批改 02 小明` | 批改第02单元，注明学生名字 |

**怎么作答**：直接在试题文件里填写：
- 选择题 → `（ B ）` 括号中填选项
- 判断题 → `（ 1 ）` 填1（对）或0（错）
- 填空题 → 替换 `____` 为答案
- 读程序题 → 在 `______` 处写结果

### 复习 — 针对薄弱点练习
| 你说 | 效果 |
|------|------|
| `复习` / `巩固` / `错题` / `查漏补缺` | 全面复习——分析所有错题出练习 |
| `复习 02` / `巩固 02` / `错题 02` | 只复习第02单元的错题 |

## 🛠️ 开始之前

1. **安装 Dev-C++**（适合新手）或 VS Code
2. 每单元的 `code/` 文件夹下有 `.cpp` 源文件，用 Dev-C++ 打开后按 **F11** 编译运行
3. 如果使用 Claude Code，以上说出题、批改、复习的功能会自动生效

---

## 💡 学习建议

1. **每天 30 分钟**，比周末猛学 3 小时效果好
2. **一定要动手敲代码**，光看是学不会编程的
3. **出错不要怕**——编译器报错是最好的老师
4. **每单元学完**说出题自测，说批改记录错题
5. **定期复习错题**，说复习生成针对性练习
