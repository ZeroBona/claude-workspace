# 钩子技术手册

## 开头钩子（前 200 字必有一项）

### 1. 冲突钩子
直接展示人物间/人物与环境/人物与自身的对抗。
```
例："把钱交出来。"刀尖抵着他的腰眼，说话的人声音很轻，像在说悄悄话。
```

### 2. 悬念钩子
利用信息不对称引发疑问。不是故弄玄虚的"他不知道即将发生什么"。
```
例：尸体被发现的时候，穿着她失踪那天的那件红色风衣。但她已经失踪三年了。
```

### 3. 反常钩子
打破读者预期的画面或事件。
```
例：法官敲下法槌的那一刻，原告席上的女人笑了。她等这个判决等了二十年——不是胜诉，是败诉。
```

### 4. 威胁钩子
即将到来的危险。
```
例：电梯里的数字一直在往下跳。手机没有信号。他已经不记得自己在地下多少米了。
```

### 5. 日常中的异常钩子
看似普通的场景中有一个不对劲的细节。特别适合都市/悬疑/日常向。
```
例：她像往常一样七点四十分出门。门口的牛奶瓶还在——两瓶。她只订了一瓶。
例：桌上的咖啡还冒着热气。但家里没人。他已经三个月没回来了。
例：微信有一条新消息。发件人是他自己。时间是三分钟后。
```

### 6. 甜文/感情向钩子
```
例：相亲对象迟到了二十分钟。他正准备走，玻璃门被推开——进来的人手里举着他的照片，照片上被人用口红画了一颗歪歪扭扭的爱心。
例：她暗恋了三年的学长，此刻正站在她面前，手里拿着她的日记本。
```

### 7. 爽文钩子
```
例："你就是那个废物？"对方话音未落，他已经一巴掌抽了过去。不是因为愤怒。是因为等这一刻，他等了整整十年。
例：系统提示音响起的瞬间，全世界都听到了。包括那些曾经嘲笑他的人。
```

## 结尾钩子（每章必有）

### 1. 悬念钩子
抛出问题但不回答。让读者脑子里自动生成"然后呢？"。
```
例：门开了。里面站着的那个人，他从未想过会再见到。
```

### 2. 反转钩子
颠覆前文认知。好的反转让读者立刻想翻回前面重看。
```
例：她一直以为自己在保护妹妹。现在才知道，妹妹一直在保护她。
```

### 3. 情绪钩子
强烈的情绪余韵让读者放不下书。
```
例（悲）：他没有哭。他把那封信折好，放进上衣口袋，贴着心脏的位置。然后继续往车站走。
例（暖）：她在口袋里摸到一颗糖。不知道他什么时候放进去的。糖纸上写了一个字：等我。
```

### 4. 威胁钩子
危险正在逼近，但角色还不知道。
```
例：他不知道那辆车已经跟了他三天。今天，它停在了他家楼下。
```

### 5. 决策钩子
角色做了不可逆的选择，暗示严重后果。
```
例："我签。"她拿起笔。窗外警笛声越来越近。但她已经没有回头路了。
```

### 6. 爽文钩子
预告即将到来的打脸/逆袭。
```
例：擂台上，裁判举起对手的手。观众席嘘声一片。没有人注意到，他嘴角的血迹下面，藏着一个笑。系统界面在他眼前展开——第二阶段，解锁。
例：他掏出那块令牌放在桌上。会议室瞬间安静。那是三年前，那个人欠他的。
```

---

## 钩子设计四原则

1. **具体**：不是一个概念（"他有了一个计划"），是一个画面（"他把地图铺在地上，在三个地点画了红圈"）
2. **当下**：不是暗示遥远的未来，是下一秒就可能发生的事
3. **与人物相关**：不是天降陨石，是角色的选择/发现/遭遇
4. **不可跳过**：读者跳过这个钩子会错过关键信息或情绪
