# 分层修改指南

## 修改优先级

修改时按以下顺序，先改影响最大的：

```
结构问题 > 人物弧光问题 > 节奏问题 > 对话问题 > 情绪表达问题 > 文笔问题
```

不要在文笔上花一小时改一段最终可能删掉的场景。

---

## 第一层：结构编辑（Developmental Edit）

**时机**：完成初稿后，逐章/逐场景审视。

**检查清单**：
- 场景存在必要性：删掉影响什么？有没有可以合并的？
- 信息释放时机：早了（读者还没兴趣）？晚了（读者已经猜到了）？漏了（读者困惑）？
- 节拍检查：催化剂/中点/一败涂地到位了吗？占比合理吗？
- 人物弧光：有变化吗？变化有铺垫吗？跳跃了吗？
- 副线：有自己的起承转合吗？和主线有互动吗？
- 节奏：高潮和平缓交替了吗？有没有连续 3 章以上没有情绪高点？

### 修改对比示例

**问题：场景没有功能**
```
❌ 修改前：
第三章，主角去菜市场买菜，和摊贩讨价还价，然后回家做饭。2000字。
—— 没有推动剧情、没有揭示人物、没有释放信息。纯日常。

✓ 修改后选项A（揭示人物）：
主角在菜市场发现有人在跟踪他。他假装讨价还价，用摊贩的镜子观察身后。
—— 日常场景变成了展示主角警觉/应对能力的舞台。

✓ 修改后选项B（释放信息）：
主角在菜市场听到两个摊贩闲聊，提到最近有人在打听他的住址。
—— 日常场景变成了信息释放的渠道。

✓ 修改后选项C（删除）：
直接跳到下一场有功能的戏。2000字省下来写真正重要的内容。
```

## 第二层：场景编辑（Line Edit）

**时机**：结构确认后，逐场景审视。

**检查清单**：
- 对话：过四层检验了吗？去掉标签能分辨说话者吗？
- 描写：具体吗？有画面感吗？感官通道够丰富吗？
- 情绪：到位吗？铺垫够吗？爆发点有力吗？
- 过渡：场景切换流畅吗？时间跳跃有标记吗？
- 节奏：该快的地方快了吗？该慢的地方慢了吗？

### 修改对比示例

**问题：对话没有辨识度**
```
❌ 修改前：
"你怎么能这样？"李明说。
"我也不想的。"王芳说。
"你知道这对我意味着什么吗？"李明说。
"我知道。对不起。"王芳说。
—— 去掉标签完全分不清谁在说话。

✓ 修改后：
李明把烟掐了。"你早就知道了。"
"知道又怎样？"王芳没看他，低头翻着手机，"告诉你，你会信吗？"
"我不信。"他把烟灰缸推到一边，"但我至少会记住——你骗过我。"
—— 去掉标签也能分辨：李明短句、攻击性；王芳回避+反问。
```

**问题：节奏拖沓**
```
❌ 修改前（200字）：
他走进房间，环顾四周。房间里很暗，窗帘都拉上了。他走到窗边，拉开窗帘，
阳光照进来，他这才看清房间的样子。书桌上放着一台电脑，还有几本书。
椅子上搭着一件外套。床头柜上有一个相框，照片里是一张全家福。

✓ 修改后（50字）：
他拉开窗帘。全家福。床头柜上的相框里，四个人——但沙发上只坐着三个。
```

## 第三层：文笔编辑（Copy Edit）

**时机**：场景确认后，逐段逐句。

**检查清单**：
- 删冗余修饰："非常/十分/极其/真的/确实/相当/蛮"
- 删重复信息：同样内容说了两遍
- "告诉"改"展示"
- 修正视角跳跃：同一段落不能换 POV
- 统一时态和人称
- 检查"的/地/得"
- 读出声来：不顺耳的地方就是需要改的地方

### 修改对比示例

**问题：情绪直接报告**
```
❌ 修改前：
他感到非常愤怒。这种被背叛的感觉让他几乎无法呼吸。
他内心涌起一股强烈的复仇欲望。

✓ 修改后：
他把那张照片从相框里抽出来。翻到背面。他前妻的字迹——"永远爱你"。
他把照片撕成两半，四半，碎片。然后去洗手。洗了很久。
```

**问题：形容词堆砌**
```
❌ 修改前：
她穿着一件绝美的、令人惊艳的红色长裙，衬托出她完美的身材和优雅的气质。

✓ 修改后：
她穿红裙子。在场所有女人都后悔了——不该和她选同一个颜色。
```

## 修改输出格式

```
【问题】[类型标签] 位置：第X段/第X行
【原文】[引用原文关键句]
【问题】[一句话说清问题]
【建议】[具体修改方向]
【示例】[改写参考，2-3句]
```

类型标签：结构/人物/节奏/对话/情绪/文笔/逻辑/设定
