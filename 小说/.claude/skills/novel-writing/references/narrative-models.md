# 经典叙事模型参考

## 模型选用指南

| 故事类型 | 推荐主模型 | 辅模型 | 原因 |
|---------|-----------|--------|------|
| 玄幻/修仙/升级流 | 英雄之旅 | 救猫咪节拍表 | 英雄之旅天然匹配"成长+跨越门槛" |
| 都市爽文/商战 | 救猫咪 | 特鲁比道德论 | 需要精确节拍控制商业节奏 |
| 言情/感情线 | 救猫咪 | 麦基鸿沟 | B故事（副线）在言情中是主线 |
| 悬疑/推理 | 麦基鸿沟 | 特鲁比 | 鸿沟推进 + 对手镜像关系 |
| 严肃文学/人物驱动 | 麦基鸿沟 | 特鲁比道德论 | 人物深度优先于情节密度 |
| 多主角/群像 | 特鲁比道德论 | — | 每个主要角色的弱点→欲望→揭示 |

**混搭原则**：取主模型的结构骨架，用辅模型的工具诊断。比如用英雄之旅搭框架，用救猫咪节拍表检查节拍是否到位。

## A. 英雄之旅（Campbell/Vogler）— 成长/冒险/玄幻

```
1. 平凡世界 → 2. 冒险召唤 → 3. 拒绝召唤 → 4. 遇见导师 →
5. 跨越第一道门槛 → 6. 考验/盟友/敌人 → 7. 接近最深洞穴 →
8. 严峻考验（死亡与重生）→ 9. 报酬（夺取宝剑）→
10. 返回之路 → 11. 复活（最终净化）→ 12. 携万能药归来
```

**网文映射**：
- 1-4 = 开局困境 + 金手指获取（1-5章）
- 5-7 = 进入新世界 + 初期打脸（5-20章）
- 8 = 第一次重大危机（20-30章）
- 9-12 = 升级回归 + 更大世界（30+章）→ 可循环

**循环设计**：玄幻/修仙通常不是一次英雄之旅，是螺旋上升的多次循环。每次循环的"平凡世界"是上一次的"新高度"，每次"万能药"是下一次的起点。

## B. 救猫咪节拍表（Blake Snyder）— 商业类型片/网文

```
1. 开场画面（1%）— 故事世界的定格
2. 主题陈述（5%）— 暗示故事要讲的道理
3. 铺垫（1-10%）— 主角的缺陷与缺失
4. 催化剂（10%）— 改变一切的事件 ★
5. 犹豫（10-20%）— 主角抗拒改变
6. 进入第二幕（20%）— 做出不可逆的决定
7. B 故事（22%）— 副线（感情/友情线）
8. 娱乐游戏（20-50%）— "预告片段落"，兑现类型承诺
9. 中点（50%）— 假胜利或假失败，赌注升级 ★
10. 反派逼近（50-75%）— 压力增大，团队瓦解
11. 一败涂地（75%）— "拂晓前的黑暗" ★
12. 灵魂黑夜（75-80%）— 主角最低谷，直面真相
13. 进入第三幕（80%）— 找到新方案/新信念
14. 结局（80-99%）— 最终对决
15. 终场画面（100%）— 与开场画面形成对比
```

★ = 必检节拍。如果这三个节拍不清晰，结构就有根本问题。

**网文适配**：
- 节奏铁律直接对应急救猫咪的百分比。2000字/章 × 100章 = 20万字：
  - 催化剂在第10章（2万字）
  - 中点在第50章（10万字）
  - 一败涂地在第75章（15万字）
- 娱乐游戏（20-50%）= 网文最好看的部分，读者入坑的关键段

## C. 麦基鸿沟理论（Robert McKee）— 严肃文学/人物驱动

核心：**人物期望与现实之间的差距 = 鸿沟**。故事在人物跨越一个个鸿沟中推进。

```
行动 → 期望结果 → 实际结果（鸿沟）→ 反应 → 新行动 → (循环)
```

鸿沟必须逐级加深：
- 第一层：外部阻碍（敌人/环境/运气）
- 第二层：自身缺陷导致的失败
- 第三层：价值观冲突——怎么做都是错

**实用方法**：每个场景写完问一句——"角色的期望是什么？实际发生了什么？差距够大吗？"

## D. 特鲁比道德论（John Truby）— 全类型通用

核心公式：**主角的弱点 → 欲望驱动 → 对抗力量 → 自我揭示**

```
弱点（心理）+ 需求（道德）→ 欲望（具体目标）→
对手（争夺同一目标）→ 计划 → 对抗 → 自我揭示 → 新平衡
```

**对手设计三原则**：
1. 和主角争夺同一个目标（不是碰巧挡路）
2. 是主角弱点的镜像（你怕什么，对手就是什么）
3. 让主角被迫面对自己的谎言

**网文应用**：大反派 = 主角如果走错路的未来版本。这让反派不是单纯的"坏人"，而是"主角的另一种可能性"。

## E. 非标准结构

不是所有故事都要线性叙事。以下结构在特定类型中有效：

### 多时间线（双线叙事）
- 过去线 + 现在线交替
- 两条线必须在情感/主题上有呼应
- 过去线的揭示必须改变现在线的意义
- 适合：悬疑/虐文/人物传记

### 环形结构
- 开头 = 结尾（但意义不同）
- 读者二刷时发现开头充满了新的含义
- 适合：悬疑/文学/时间循环

### 碎片叙事
- 多个看似无关的视角/事件 → 逐渐汇聚
- 每个碎片必须有独立的价值（不能只是拼图的一块）
- 适合：群像/灾难/战争

### 使用原则
非标准结构增加写作难度和阅读门槛。网文慎用（平台算法不友好）。如果要用，必须在开篇就给读者一个"锚点"（一个能抓住的线索或情感）。
