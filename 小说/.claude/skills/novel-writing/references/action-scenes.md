# 动作场景指南

## 通用原则

### 视角
紧贴 POV 角色的感官：他看到的/听到的/感觉到的。不要上帝视角。

```
❌ 对手的刀从左侧劈来，身后还有两个人正在逼近，远处弓箭手已经瞄准。
   → POV 角色不可能同时看到身后和远处

✓ 刀锋劈下来。他侧身避开，肩膀擦着刀刃过去。身后有脚步声——不止一个。
```

### 代价原则
每场动作后必须有代价：
- 身体代价：受伤/消耗/残疾
- 资源代价：武器损毁/弹药耗尽/失去装备
- 心理代价：创伤/恐惧/负罪感
- 关系代价：暴露秘密/失去信任/牺牲他人

无代价的动作场景 = 回合制游戏。

## 战斗场景

### 节奏公式
```
短句轰炸（5-10句） → 慢镜头细节（1-2句） → 短句轰炸（5-10句） → 后果静默（1-3句）
```

### 禁忌

**❌ 回合制**：战斗不是你一下我一下。是混乱、同时发生、信息过载和瞬间判断。

**❌ 战斗中长篇解说**：招式名和原理留给战斗前或战斗后的复盘。

**❌ 敌人排队**：真实的群战是同时围攻。主角用环境/走位/策略应对数量劣势。

**❌ 主角突然觉醒秒杀**：觉醒必须有铺垫（训练/代价/伏笔）。

### 优秀战斗的要素
1. **明确赌注**：输了会怎样？（不只是"会死"，要具体）
2. **环境参与**：不只是拳脚，是用一切能用的东西
3. **策略感**：不是靠蛮力，是靠判断和选择
4. **情绪内核**：不只是动作，是愤怒/恐惧/绝望/决心
5. **后果延续**：这场战斗的代价影响后续剧情

## 追逐场景

### 节奏公式
```
环境切换 → 障碍克服 → 距离变化 → 意外 → 环境切换 → (循环，逐级升级)
```

### 关键要素
- **空间感**：读者要能想象追逐的空间关系（谁在前谁在后/距离多远）
- **障碍升级**：每次障碍比上一次更难
- **创造性**：不只是跑，是利用环境/工具/人群/信息差
- **视角优势**：追的人和逃的人看到的信息不同（制造悬念）

### 示例
```
他撞开安全门冲进小巷。垃圾桶、晾衣绳、堆叠的纸箱——他跳过、钻过、踢翻。身后的脚步声越来越近。
前面是死胡同。
一堵三米高的墙。没有助跑距离。
他回头看。追他的人也停下了。两个人隔着十米喘气。
"你跑不掉的。"
他没回答。他的手在身后摸到了一根生锈的水管。
```

## 潜入/潜行场景

### 核心张力来源
- 被发现的风险（每时每刻）
- 信息获取与代价的权衡（多走一步可能看到关键信息，也可能暴露）
- 时间的压迫感

### 节奏公式
```
规划 → 执行（顺利）→ 第一次意外（小）→ 调整 → 重大意外（濒临暴露）→ 惊险过关 → 获取关键信息/到达目标 → 新的意外（结尾钩子）
```

### 关键技巧
- 用环境音和光线营造氛围
- 每一次"差点被发现"都要比上一次更险
- 沉默中的心理活动（但不能长篇大论）
- 读者和角色一起屏住呼吸

## 竞技/比赛场景

### 与战斗的区别
- 有规则（可以打破规则但要有代价）
- 有观众（观众反应是重要的情绪放大器）
- 有时间/比分压力

### 节奏公式
```
赛前状态（劣势/被轻视）→ 开局不利 → 第一次调整 → 扳回一城 →
第二次危机（更严重）→ 突破极限 → 决胜时刻 → 赛后余波
```

### 关键技巧
- 不要让主角一路碾压（没有悬念）
- 对手必须有真实实力（赢才值得）
- 观众/解说的反应穿插在动作之间（调节节奏）
- 胜负不只是比分，是尊严/信念/关系的证明

## 情感动作

不是所有动作场景都是打打杀杀。有时候最强的动作场景是情感爆发。

### 情感动作 = 动作 + 情绪重量

```
❌ 她哭了。（告诉）
✓ 她把相框扣在桌上。然后一个接一个，把房间里所有的相框都扣了过去。最后一个扣完，她才蹲下来，把脸埋在膝盖里。（展示）
```

### 关键
- 用具体的、可见的动作承载情绪
- 动作的选择要体现人物性格（不同人表达悲伤的方式不同）
- 节奏：压抑 → 失控 → 宣泄 → 空白的静默
