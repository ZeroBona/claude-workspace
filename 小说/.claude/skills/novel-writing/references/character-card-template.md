# 人物卡模板

## 完整模板（主角及重要配角使用）

```
【姓名】+ 外号/称号 + 名字的含义或来历（如有）
【核心标签】3 个关键词（如：腹黑、护短、嗜甜）
【年龄/身份/处境】现在的社会位置 + 所处的困境
【表层欲望】想要什么（外驱力，具体可感知）
【深层需求】缺什么（内驱力）— 角色自己通常不知道
【致命缺陷】性格/能力的致命短板
【核心谎言】角色相信的错误信念
   例："只要足够强，就不会再失去任何人"
   例："只要不付出真心，就不会受伤"
   例："只要够聪明，就能控制一切"
【核心真相】角色最终学到的东西
   例："真正的强大是敢于示弱"
【弧光轨迹】
   从[状态A] → 催化事件[具体事件] → 中点转折[事件] → 一败涂地[事件] → 觉醒[顿悟] → [状态B]
【说话方式】
   句式长短：短促（5-10字）/ 中等 / 冗长
   用词档次：粗俗 / 日常 / 文雅 / 专业术语多
   口头禅：
   语速感：快（连珠炮）/ 中 / 慢（字斟句酌）
   攻击性：高（句句带刺）/ 中 / 低（回避冲突）
   特色：方言/口音/外语夹杂/爱用反问/爱用比喻
【标志动作】一个反复出现的小动作
   例：摩挲戒指/咬嘴唇/转笔/摸后颈/整理袖口/反复解锁手机
   用法：紧张时频率增加，放松时消失。不同情绪下动作的质感不同。
【关键台词】一句最能体现角色本质的话（读者能凭这句话认出角色）
【关系网络】
   - 与[角色A]：[关系类型] + [核心冲突/利益纽带] + [关系的起点→终点变化]
   - 与[角色B]：...
【功能定位】主角/反派/导师/伙伴/爱情对象/捣乱者/镜子角色/催化剂
【原型参考】英雄/智者/魔术师/反叛者/爱人/小丑/凡夫/守护者/创造者/统治者
【原型阴影】这个原型的负面/失衡形态是什么？
   例：英雄的阴影是 bully（欺凌者）；守护者的阴影是控制狂
```

## 简化版（次要配角用）

```
【姓名】+ 外号
【核心标签】2-3 个词
【欲望+缺陷】一句话
【说话特征】2 个关键词
【关系】和主角/关键角色的核心冲突
【功能】在剧情中起什么作用
```

## 人物关系图绘制方法

### 方法一：冲突-利益矩阵

```
        利益一致        利益冲突
冲突    "友军"          "死敌"
关系    （合作但有矛盾） （你死我活）
        "盟友"          "对手"
无冲突  （完全一致）     （竞争但不敌对）
关系
```

把每个重要角色放进对应的象限，确保关系多样化。所有人都挤在一个象限 = 关系单一。

### 方法二：人物网检查

对每个角色问：
1. 他和主角的关系是什么？（利益 + 冲突）
2. 他和至少两个其他配角的关系是什么？
3. 他的存在改变了谁？（如果没有改变任何人 = 工具人）
4. 如果删掉他，剧情哪里会断？（断不了 = 可以删）
