# 老宅 — 人物卡

## 周德厚（老大）

【年龄】52岁
【身份】农民，一辈子没离开过村子
【核心标签】沉默、倔强、不会表达
【一句话】"伺候了十年，到头来他一个字都没跟我说。"

- 初中没毕业就下地干活，供弟弟妹妹读书
- 娶了本村女人李秀琴，有两个孩子，都在外省打工，一年回来一次
- 父亲最后十年，他每天过去做饭、洗衣、照顾，做完就走。晚上回自己家
- 但他从来没跟父亲好好聊过天。两个人坐在一起，一晚上可以不说一句话
- 他心里有怨气，但从来不说出口。他觉得说出来就是"不孝"
- 去年在县城撞见父亲从养老院出来，父亲说看一个老朋友，求他保密。他没追问，也不敢再去——他怕看到自己不想看的
- 父亲去世时，他的两个孩子请不了假，没能回来奔丧
- 长得像父亲。第8集林秀兰把他错认成周青山，第24集再次错认
- 第19集：学母亲说话的语气，把老三逗笑了——这是他第一次展现柔软的一面
- 第23集：在母亲坟前，第一次在弟妹面前掉了眼泪
- 第30集：林秀兰叫了他的名字，不是父亲的名字

【弧光】从"觉得自己牺牲最多"→ 发现自己只是在"待着"→ 学会表达 → 接过父亲的遗愿 → 被林秀兰真正记住

【关键台词】
- "十年，三千六百五十天。你们谁回来过？"
- "他最后那几天，我在地里。我以为他还能撑几天。"
- "爸没做完的事，我们帮他做完。"
- "有些账，不是用钱算的。"
- "妈要是知道了，她不会怪爸的。她那个人，从来不怪别人。"

【评论区定位】读者心疼他、替他委屈，但也会有人觉得他"太计较"

## 李秀琴（大嫂）

【年龄】50岁
【身份】老大的妻子，农村妇女
【一句话】"他心里只有他爸，什么时候有过这个家？"

- 嫁给老大三十年，吃苦耐劳，但心里有怨气
- 她觉得老大对父亲比对自己家上心——但最后收拾老宅时，她也在，哭得比谁都厉害

【功能】第13集电话冲突制造者，代表"普通人的正常反应"

## 周德利（老二）

【年龄】48岁
【身份】县城建材店老板，有一子一女
【核心标签】精明、现实、嘴巴硬
【一句话】"我以为钱能解决的事，都不叫事。"

- 二十年前离开村子，白手起家
- 每年给父亲打两万块，觉得自己"该做的都做了"
- 父亲去世那天，他正在谈一笔二十万的生意
- 他其实不是不孝，他只是觉得"日子还长"
- 有一个15岁的女儿和一个12岁的儿子，父亲去世时正在考试，没让回来
- 第9集：追问王婶"爸为什么现在才去找她"——王婶答"你们妈走了以后"
- 第10集：主动算账，说出金句
- 第13集：去银行查流水，发现父亲在他打钱的后一天把钱存进账户
- 第21集：取消生意，决定整理父亲的账本
- 第24集：在走廊里透过门缝看老大陪林秀兰，转过身靠在墙上闭眼
- 第28集：主动提出每月出两千

【弧光】从"精明算计"→ 发现亏欠的不是钱而是时间 → 取消生意 → 成为父亲的"账本守护者"→ 主动承担

【关键台词】
- "我一年打两万，十年二十万。我以为够了。"
- "他从来没跟我们提过一个字。"
- "我爹的事，还没办完。"
- "他攒的不是钱，是去见她的路。"
- "爸给每个人都准备了东西。他只是一样都没给出去。"

## 周晓梅（老三）

【年龄】43岁
【身份】远嫁广东，家庭主妇，有一子在外地上大学
【核心标签】隐忍、愧疚、不敢回家
【一句话】"我不是不想回来，我是回不来。"

- 二十岁嫁到广东，距今二十三年
- 丈夫是工厂老板，控制欲强，不让她回娘家
- 父亲去世前三个月，她偷偷打过一次电话，父亲说"没事，爸好着呢"。她听出声音不对，但选择相信
- 穿貂皮大衣回来，不是炫耀——是怕被人看出她过得不好
- 第12集：在车上第一个站出来为林秀兰说话——"她不是外人，她陪了爸最后三年。我们呢？我们谁陪了？"
- 第15集：第一个说"我同意大哥"
- 第22集：反抗丈夫后，主动提出去看母亲的坟——十五年没去过了
- 第27集：在母亲首饰盒里发现父亲二十三年前准备的嫁妆金项链
- 第30集：回去了，但丈夫的态度变了——她在家说话，他开始听了

【弧光】从"懦弱隐忍"→ 第一个为林秀兰说话 → 第一个对丈夫说"不"→ 给母亲扫墓 → 找回尊严。不是离婚，是让丈夫学会尊重她。

【关键台词】
- "我每次说要回来，他就说'你爸妈又不需要你'。"
- "爸跟我说'没事，好着呢'。我信了。我明明听出他声音不对，但我信了。"
- "她不是外人，她陪了爸最后三年。我们呢？我们谁陪了？"
- "二十年前我走的时候，爸站在村口。我回头看了一眼，他还在那儿。"
- "我先不回去，把我爸的事办完。"
- "大哥，我想去看看妈。妈的坟，我十五年没去过了。"

## 刘淑珍（母亲，已故）

【年龄】享年58岁，去世十五年
【身份】周青山的妻子，三个孩子的母亲
【一句话】"她走了以后，这个家就再也没有人张罗过年了。"

- 和周青山相亲认识，结婚三十八年
- 勤俭持家，把三个孩子拉扯大
- 她和周青山感情实在，是那种一起过日子、一起吃苦的伴
- 她不知道林秀兰的存在。周青山从来没提过
- 她去世前拉着老大的手说：照顾好你爸，他这个人闷，心里有事不说
- 她的性格：从来不怪别人。第19集老大说"妈要是知道了，她不会怪爸的"
- 第23集：三个孩子终于一起去给她扫墓

【功能】不在场的对照。她的存在证明周青山对得起婚姻——婚内没有背叛，但心里始终有一个角落不属于她。

## 周青山（父亲，已故）

【年龄】享年73岁
【身份】老农民，一辈子没出过县城
【一句话】"他沉默了一辈子，死后才让人知道他在想什么。"

- 年轻时爱过林秀兰，因为成分问题被迫分开
- 后来经人介绍，和隔壁村的刘淑珍相亲结婚。三十八年夫妻，相敬如宾
- 妻子在世时，从未联系过林秀兰。一次都没有
- 妻子去世后第三年，打听到林秀兰进了养老院，无儿无女没人管
- 每月坐两小时班车去看她，风雨无阻，三年三十六个月，一次没断过
- 攒了三十万——大部分是妻子去世后攒的，一个人没什么开销，一分一毛抠出来的
- 去年在县城被老大撞见，跟老大说看一个老朋友，求老大保密
- 有一本旧笔记本，记录林秀兰病情，夹在养老院椅子缝里。最后一页写于去世前一天：明天下雨，去不了。等她
- 给每个孩子都写过信，一封都没寄出去
- 给老三准备了嫁妆金项链，压在箱底二十三年
- 给每个孩子都准备了东西，一样都没给出去
- 对三个孩子从来没当面夸过——但在林秀兰面前，把每个孩子都夸了一遍
- 老二每月打钱的后一天，他就把钱存进那个账户

【功能】不在场的主角。通过子女的回忆、信件、笔记、账本、他人的叙述，拼凑出完整形象。

## 林秀兰

【年龄】71岁
【身份】周青山年轻时的恋人，养老院老人
【一句话】"她认不出任何人，但她记得周青山。"

- 年轻时和周青山相恋，因成分问题被迫分开
- 嫁了个不爱的男人，丈夫早逝，无儿无女
- 晚年痴呆，被远房亲戚送进养老院后再也没人来看过
- 周青山每次来都带同样的搪瓷缸子。她看到老大（长得像父亲）拿着搪瓷缸子，条件反射地笑了
- 第24集：她把老大错认成周青山
- 第25集：清醒一瞬，断断续续说出父亲夸三个孩子的话。说到最后又糊涂了
- 第30集：叫了老大的名字——"德厚"。不是"青山"，是"德厚"

【功能】故事的情感锚点。最后叫出老大的名字，说明老大不再只是"父亲的影子"。

【关键台词】
- "青山，你来了。"
- "青山说……德厚像他，不会说话。德利……太忙。晓梅……像她妈，爱笑。"

## 王婶

【年龄】65岁
【身份】周家邻居，村里住了五十年
【一句话】"有些事，老头子不说，但我都看在眼里。"

- 嘴碎，但心不坏
- 是村里唯一知道周青山和林秀兰年轻往事的人——她和周青山同村，五十年前的事都看在眼里
- 在县城医院见过周青山一次，才知道他找到了林秀兰。周青山求她保密
- 第9集：三兄妹你一句我一句地问，她断断续续讲出真相。老二追问"爸为什么现在才去找她"，王婶答"你们妈走了以后"

【功能】第9集真相释放者，触发第10集算账
